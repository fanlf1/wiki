# AR 眼镜光波导显示技术问答

> 来源：从 Even Realities G1 眼镜出发，追问 MicroLED + 光波导的显示原理。2026-05-27。

## 背景

Even Realities G1/G2 采用 **JBD（显耀显示）单绿 MicroLED + 衍射光波导**方案。追问了三个递进问题：

---

## Q1：镜框处的光源投射与 MicroLED 是不同方案吗？

**不是，是同一套方案的三个环节。**

```
镜腿/铰链处                    镜片                     眼睛
┌─────────┐               ┌──────────────┐          ┌─────┐
│ JBD      │  ──光束──>   │ 衍射光波导    │ ──衍射──> │ 👁  │
│ MicroLED │    耦合入     │ (透明镜片)    │    出射   │     │
│ 微显示屏  │    波导边缘   │ 内嵌衍射光栅  │          │     │
└─────────┘               └──────────────┘          └─────┘
  ↑ 你看到的光源              ↑ 透明镜片
  这就是 MicroLED 本身        这就是光波导
```

| 组件 | 位置 | 作用 |
|------|------|------|
| MicroLED 芯片 | 铰链/镜尾处 | 光源 + 图像生成 |
| 耦合光栅 | 波导镜片边缘 | 将光"灌入"镜片 |
| 出射光栅 | 波导镜片正面 | 将光"导出"到眼睛 |

MicroLED 不是一种"投射方式"，而是**显示芯片技术**（类似 OLED、LCD，但用微米级 LED 作为像素）。它配合光波导才能实现"透明镜片上显示画面"。

---

## Q2：光波导如何投射出平面画面的像素？

核心原理：**衍射光栅**在镜片内部"折叠"光路，精确地把每个像素的光导向眼睛。

### 三层机制

**1. 全反射（光被"关"在玻璃里）**
光以大角度射入玻璃后，在上下表面之间不断反射，跑不出去。镜片看起来透明——光在里面走，外面看不到。

**2. 衍射光栅（纳米级凹槽打破全反射）**
玻璃表面刻了纳米级周期性凹槽（间距约几百纳米，接近光的波长）。光每经过一次凹槽，就有一部分被衍射出玻璃表面。

**3. 成像原理（角度 → 空间位置映射）**

```
MicroLED芯片上：           波导出射后你看到的：

  ■ ■ ■ ■ ■               P1  P2  P3  P4  P5
  像素阵列                   ← 对应不同空间位置
  每个像素发出的光
  角度略有不同

不同角度的光 → 在波导中弹射次数不同 →
在出射光栅的不同位置被衍射出来 →
形成空间上分开的像素点 → 平面图像
```

光波导本质上是一个**角度→空间位置的映射器**。它不是自己在发光，而是在**精确控制光从哪里漏出来**。

## Q3：绿光波长与波导厚度的关系？

### JBD 绿光 MicroLED 参数

| 参数 | 值 |
|------|-----|
| 波长 λ | **~530 nm** |
| 光栅周期 Λ | **270-400 nm** |
| 波导折射率 n | 1.7-2.0 |
| 波导厚度 | ~1 mm |

### 精确的比例关系

比例关系存在于**光栅周期和波长**之间，而非波导厚度和波长之间：

```
光栅方程：Λ = λ / (n_eff × Δsinθ)

光栅周期 Λ ≈ λ/n（与波长同一量级，~300nm）
波导厚度   ~1mm（远大于波长，约2000倍λ，由工程折中决定）
```

| 设计参数 | 决定因素 | 与波长的关系 |
|----------|----------|-------------|
| 光栅周期 Λ | 光的衍射条件 | Λ ≈ λ/n，精确相关 |
| 波导厚度 d | 机械强度 + 全反射条件 + 量产工艺 | d >> λ，工程参数 |

这就是为什么单色绿光（如 Even G1）的光波导比全彩容易做——只需为一个波长优化一组光栅周期。全彩（R+G+B）需要同时满足三个波长（~630nm, ~530nm, ~450nm）的衍射条件，光栅设计复杂度指数级上升。

# Tencent腾讯投资

## 一句话定位
> 腾讯投资是全球最活跃的CVC之一，2024-2025年战略重仓医疗健康/创新药赛道，约40%投资事件投向医疗，被称为"医疗IPO超级捕手"。

## 基本信息

| 项目 | 内容 |
|------|------|
| 英文名 | Tencent Investment |
| 类型 | CVC（企业风险投资） |
| 所属公司 | 腾讯控股（HKSE: 0700） |
| 总部 | 深圳 |
| 领导人 | 刘炽平（Martin Lau，腾讯总裁，全面负责战略与投资） |
| 投资组合规模 | ~RMB 9,483亿元（截至2025年6月） |
| 投资组合公司数 | 600+家 |
| 医疗健康占比 | ~11%（历史），2024年至今占比升至~40% |
| 证据力评级 | B级（补充信号） |

## 与我们的关系
- **关系类型**：赛道信号验证（医疗健康大方向）+ 潜在行业情报
- **关注优先级**：中
- **核心关注点**：腾讯在医疗健康赛道的投资重心是创新药/AI制药，与我们的CGM/智能戒指/BCI赛道交集有限，但其战略转向医疗本身是赛道热度的信号

## 投资哲学与逻辑

### 核心理念
1. **战略投资优先**：李朝晖公开表示"一个具有战略价值的好项目，最终也一定会包含财务价值"
2. **科技赋能**：利用腾讯的AI、大数据、云计算（微信生态+腾讯云）为被投企业赋能
3. **生态构建**：在医疗领域目标是整合医生、患者、药企进入腾讯技术生态

### 医疗投资逻辑演变

| 阶段 | 时间 | 策略 | 重点 |
|------|------|------|------|
| 第一阶段 | 2014年前后 | "连接型"扫货 | 互联网医疗平台（丁香园、微医、好大夫等），以流量为纽带搭建连接平台 |
| 第二阶段 | ~2018年 | 延伸纵深 | 医疗大数据、AI辅助诊断、医疗器械，布局健康保障（水滴、思派） |
| 第三阶段 | 2023至今 | 重注创新药 | AI制药、创新药研发，"广撒网+验证后入局"策略降低R&D风险 |

### 投资策略特点
- **广撒网+验证后入**：在创新药高风险领域，投资多家公司分散风险，通常在PoC（概念验证）后、大药企BD活跃时进入
- **偏好早期**：天使轮或B轮进入后持续加注
- **大换仓**：2024-2025年主动从低增长领域退出，将资本重新配置到高确信度医疗/创新药方向

### 自有技术平台
- **腾讯云深智药**（2020年上线）：AI药物发现平台
- **腾讯医疗大模型**（2023年上线）：面向医疗场景的AI大模型，已落地1300+家机构

## 最新动态与信号

### 战略转向（2024-2025）
- **重仓医疗健康**：2024年至今22起投资中9起在医疗健康，占比~40%，超过AI赛道
- **创新药为主**：8起医药投资覆盖创新药研发、超声影像、早筛检测
- **2025年开年大手笔**：4起过亿投资中2起在医疗/生物科技
- **"大换仓"**：主动从低增长领域退出，资本重配到高确信度医疗方向

### 2026年展望
- 被称为腾讯"医疗IPO收获大年"，多家被投企业排队上市
- AI+药物发现仍是重点（云深智药平台+医疗大模型）

### 关键信号判断

| 信号 | 解读 |
|------|------|
| 腾讯40%投资投向医疗 | 医疗健康作为投资赛道得到顶级CVC验证，但重心在创新药而非器械/穿戴 |
| 腾讯未投CGM/智能戒指/BCI | 这些细分赛道在腾讯投资视野中优先级不高，或尚未看到满意标的 |
| 腾讯投资医疗以"连接型"为主 | 其核心逻辑是流量+平台，与我们的硬科技路径有差异 |
| 与圣湘/菲鹏战略合作 | 显示腾讯在诊断/检测领域的兴趣，但更多是平台合作而非股权投资 |

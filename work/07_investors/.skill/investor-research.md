# 投资机构深度调研技能

> 从 a16z Bio+Health 深度调研中归纳，经 22 家机构批量调研验证和迭代的方法论。适用于所有投资机构的广度+深度调研。

## 调研框架

一次完整的投资机构调研 = **6 个模块**，按顺序执行：

### 模块 1：基金规模与历史

**目标**：理解机构的资金体量、成长轨迹、当前状态。

| 信息项 | 获取方式 | 关键判断 |
|--------|----------|----------|
| 各期基金规模、年份 | 官网、Crunchbase、PitchBook | 规模递增=市场认可，缩减=信号转变 |
| 总 AUM | 行业报告、新闻 | 体量决定影响力 |
| 最新募资状态 | 新闻搜索 `"<机构名> fund <年份>"` | 募集中=仍在出手，缩减=可能收缩 |
| 关联/联合基金 | 官网公告 | 如 Lilly 联合基金模式，反映新策略 |
| 重大人事变动 | 新闻搜索 | 创始GP离任=转折点 |

**搜索关键词**：
```
"<机构名> fund size history"
"<机构名> fund I II III IV" 
"<机构名> AUM assets under management"
"<机构名> new fund <年份>"
"<机构名> partner leaves <年份>"
```

### 模块 2：完整投资组合

**目标**：全景式列出所有被投公司，按赛道分类。

**获取方式**：
1. **官网 portfolio 页面**（最权威，如 a16z.com/investment-list）
2. **Crunchbase 组织页**（含轮次、金额）
3. **CB Insights / PitchBook**（更详细，需付费）
4. **官网 blog/新闻**（最新投资通常先在 blog 宣布）

**分类方法**（按我们的赛道体系）：
```
数字健康 / AI+医疗 / 代谢健康 / 消费级诊断
可穿戴 / 生物传感 / 药物发现 / 基因组学
其他（医疗IT、医疗Fintech等）
```

**每家公司记录**：
| 字段 | 说明 |
|------|------|
| 公司名 | — |
| 做什么 | 一句话 |
| 轮次/金额 | 尽可能精确 |
| 时间 | 年份 |
| 领投GP | 哪位合伙人主导 |
| 备注 | 与我们的关系、退出状态等 |

**搜索关键词**：
```
"<机构名> portfolio companies list"
"<机构名> investments <年份>"
"<机构名> <赛道> investments"
```

### 模块 3：关键合伙人团队

**目标**：识别决策者，理解投资偏好。

| 信息项 | 用途 |
|--------|------|
| 姓名、职位 | 跟踪公开言论 |
| 关注方向 | 判断投资偏好 |
| 教育背景 | 理解认知框架 |
| 前雇主/创业经历 | 判断行业洞察深度 |
| 代表投资 | 哪些公司是这个人的"代表作" |

**特别关注**：
- GP变动（加入/离任）是重要信号
- 每个GP通常有2-3个核心关注领域
- 新加入的GP可能带来方向转变

**搜索关键词**：
```
"<机构名> general partners team"
"<机构名> GP bio profile"
"<合伙人名> investments background"
```

### 模块 4：退出记录

**目标**：判断投资回报能力和投资周期。

| 退出类型 | 关注点 |
|----------|--------|
| IPO | 上市时间、交易所、当前市值 |
| SPAC | 质量参差，注意后续表现 |
| 收购 | 被谁收购、估值 |
| 已关闭 | 失败案例，理解失败原因 |

**关键指标**：
- 退出总数 vs 投资总数（退出率）
- 平均投资到退出周期（判断耐心资本）
- 退出估值分布（高回报 vs 平平）

**搜索关键词**：
```
"<机构名> exits IPO acquisitions"
"<机构名> portfolio IPO"
"<机构名> successful investments returns"
```

### 模块 5：投资逻辑/论文

**目标**：理解他们为什么投，而不是投了什么。

**信息来源**：
1. **官网 Blog**（最丰富，a16z 的 blog 是行业标杆）
2. **播客**（投资逻辑的深度表达）
3. **Newsletter**（持续更新的观点）
4. **公开演讲/访谈**（YouTube、会议）
5. **LP 报告**（如有泄露）

**归纳方法**：
- 找到 3-5 篇核心论点文章
- 提炼投资主题（通常 3-5 个）
- 注意观点随时间的变化（如从"Bio"到"AI+Health"的重心转移）

**搜索关键词**：
```
"<机构名> investment thesis"
"<机构名> bio health strategy"
"<机构名> podcast newsletter"
"<合伙人名> blog posts"
```

### 模块 6：最新动态与趋势

**目标**：捕捉近期变化，判断方向调整。

| 关注点 | 获取方式 |
|--------|----------|
| 最新投资 | 官网 blog、Crunchbase 筛选 |
| 基金规模变化 | VC 行业媒体（TechCrunch, VCJ） |
| 人事变动 | LinkedIn、行业新闻 |
| 观点文章 | 官网 blog、Substack |
| 联合投资方 | Crunchbase syndicate 分析 |

## 调研执行顺序

```
1. 搜索基金规模 → 建立"实力基线"
2. 搜索完整投资组合 → 建立"广度全景"
3. 搜索GP团队 → 建立"决策者画像"
4. 搜索退出记录 → 建立"成绩单"
5. 搜索投资逻辑 → 建立"为什么"
6. 搜索最新动态 → 建立"现在在做什么"
7. 综合分析 → 写入档案 + 更新 overview.md
```

## 信号解读规则

### 从投资组合看赛道信号

| 模式 | 解读 |
|------|------|
| 同一赛道投多家 | 强信号：机构对该赛道有 conviction |
| 只投一家就停 | 弱信号：可能是试水，观察后续加注 |
| 从 Seed 连续加注到 C 轮 | 最强信号：全程领投=极高信心 |
| 短期密集出手 | 赛道窗口期，FOMO 驱动 |
| 长期布局（3年+） | 深思熟虑的 conviction |

### 从基金变化看方向调整

| 模式 | 解读 |
|------|------|
| 新基金规模扩大 | 市场看好，出手会更频繁 |
| 新基金规模缩减 | 市场承压，投资更集中精选 |
| 创始GP离任 | 方向可能调整，关注继任者偏好 |
| 新增联合基金 | 新的合作模式/方向（如 Lilly 联合基金） |

### 从GP行为看微观信号

| 模式 | 解读 |
|------|------|
| GP频繁写某赛道文章 | 该方向是其当前核心 conviction |
| GP在播客讨论某话题 | 试探性观点，可能预示投资方向 |
| GP的背景与某赛道匹配 | 投资偏好可预测（如 MD+PhD 的 GP 更倾向诊断/临床） |

## 与 overview.md 的联动

每完成一家机构的深度调研后：
1. 更新 overview.md 中该机构的条目
2. 评估证据力等级是否需要调整
3. 检查赛道信号总览是否需要更新
4. 新发现的投资事件补充到「投资事件明细」
5. 如涉及新被投公司，考虑在 06_brands/ 或 05_vendors/ 创建文件

## 按机构类型的差异化策略

> **教训**：不同类型的投资机构，数据可获取性差异巨大。a16z 的信息透明度是特例，不是常态。

### VC（独立风险投资）

- **数据源**：Crunchbase、官网 portfolio、Blog/Newsletter
- **典型问题**：信息通常较公开，英文资料丰富
- **代表**：a16z、Khosla、红杉、启明

### CVC（企业战投）

- **数据源**：母公司公告、工商变更、被投公司融资新闻、36氪/投资界
- **典型问题**：通常无独立官网，投资披露不规律，战略投资vs财务投资需区分
- **搜索特有关键词**：
  ```
  "<母公司名> 战略投资" "<母公司名> 产业基金"
  "<母公司名> 投资 收购 <赛道>" "<CVC名> portfolio"
  ```
- **代表**：华为哈勃、小米/顺为、OPPO、vivo、腾讯、美团龙珠

### 国资/政府背景基金

- **数据源**：官网（通常有）、政府公告、中国基金业协会、证券时报/投资界、地方金融局公告
- **典型问题**：基金规模常以"认缴规模"而非"实缴"披露；投资组合公开度低，需从被投公司端反查
- **搜索特有关键词**：
  ```
  "<机构名> 投资 医疗" "<机构名> 基金规模 认缴"
  "<机构名> 引导基金 母基金" "site:pedaily.cn <机构名>"
  ```
- **特别关注**：国资基金的政策导向性（"国家需要什么就投什么"），人事变动可能涉及纪检调查
- **代表**：深创投、国投创合、中金资本、招商局资本、中移资本

### PE/大型综合基金

- **数据源**：官网、年报（如上市）、行业报告、WSJ/FT（国际化基金）
- **典型问题**：投资组合庞大（数百甚至上千），需聚焦筛选；管理规模可能有多个口径
- **代表**：高瓴、华润资本

## "无投资"也是重要结论

> **教训**：调研后发现某机构在我们核心赛道（CGM/戒指/BCI/生物传感/可穿戴）无直接投资，这本身是有价值的结论，不是调研失败。

**处理方式**：
1. 在 investor 页面明确标注"该赛道暂无直接投资"
2. 评估是否与机构投资逻辑一致（如微软M12不投硬件是合理的）
3. 判断是否有间接信号（如英伟达投AI制药验证了AI+健康赛道热度）
4. 审视 evidence_tier 是否需要下调（核心赛道无投资=信号弱）

**评级重估规则**：

| 调研发现 | tier 调整 |
|----------|----------|
| 核心赛道有≥2家投资 | 维持或上调 |
| 核心赛道有1家投资 | 维持 |
| 核心赛道无投资，但有相关赛道投资 | 维持，标注"间接信号" |
| 核心赛道无投资，且逻辑上不太可能投 | 考虑下调 |
| 发现新的重要投资动作 | 上调 |

## 批量调研执行规范

### 并行调研策略

批量调研多家机构时，**禁止一次性启动超过 4 个 agent**，避免 API 限频导致全部失败：

```
正确做法：
  第1批：启动 3-4 个 agent → 等待完成
  第2批：启动下一批 3-4 个 → 等待完成
  ...

错误做法：
  一次性启动 8 个 agent → 全部触发限频 → 浪费时间
```

### status.json 同步检查（必做）

调研前**必须先读取文件内容**确认是否真的需要更新，不能只看 status.json：

```
1. 读取 status.json → 找出 update_status != "已完成" 的机构
2. 对每个机构，读取其 .md 文件，检查缺失模块是否真的缺失
3. 如果文件已包含该模块但 status.json 未更新 → 直接更新 status.json，跳过调研
```

**Why:** 曾发现华为哈勃和联想创投的 .md 文件已包含完整 exits + depth，但 status.json 仍显示 5/7，导致重复认领和无效工作。

### 状态管理流程

```
调研前：status.json 设为 "进行中" + 具体说明（如 "进行中（补exits）"）
调研中：正常执行
调研后：status.json 改为 "已完成"，同步更新 modules 和 completeness
```

### CVC 机构的 exits 模块预期管理

对于投资活动极少的 CVC（如 vivo、OPPO），exits 模块的结果天然有限。这类机构的 exits 模块**不需要深度调研**，简要说明"公开退出数据极少"并列举 1-2 个已知案例即可。

判断标准：
- 机构明确以"自研为主"而非投资驱动 → exits 模块简化处理
- 对外投资少于 10 家 → exits 基本可预判为"数据有限"
- 非上市公司 CVC → 不存在通过自身 IPO 退出的路径

### 模块完成度目标与机构等级的关系

| 机构等级 | 目标完成度 | 说明 |
|----------|-----------|------|
| S 级 | 7/7 ★ | 需要完整的 depth 深度调研 |
| A 级 | 7/7 ★ | 需要完整的 depth 深度调研 |
| B 级 | 6/7 | depth 可选，其余模块必须完成 |
| C 级 | 6/7 | depth 可选，其余模块必须完成 |

**Why:** depth 模块需要完整的 6 模块深度调研，对 B/C 级机构（与核心赛道距离较远）的投入产出比不高。

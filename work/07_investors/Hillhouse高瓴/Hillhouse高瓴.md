## 4. thesis：投资逻辑/论文

### 4.1 核心投资哲学

| 原则 | 阐述 | 典型案例 |
|------|------|----------|
| **长期主义——"做时间的朋友"** | 拒绝短期投机，穿越周期，陪伴优秀企业长期成长 | 百济神州6年8次加仓（2015-2022），后理性退出 |
| **守正用奇** | 坚守价值投资原则（守正），同时拥抱变化、灵活应对（用奇） | 早期投腾讯+后期$44亿收购飞利浦家电，跨越全周期 |
| **深度价值创造（DVC）** | 不限于财务投资，建立DVC团队深度参与被投企业运营管理 | 百丽国际私有化后数字化转型；微创医疗对赌条款驱动减亏 |
| **与伟大格局观者同行** | 重视企业家愿景和执行力，"投资就是投人" | 蓝月亮10年陪伴（唯一外部投资者） |

### 4.2 医疗健康投资论

1. **全产业链布局**：系统性覆盖CXO→创新药→医疗器械→数字健康→医疗服务→医药零售全链条
2. **"中国版美敦力"路径**：在医疗器械领域追求控制权，以"创新+整合+出海"三步走，打造中国版美敦力
3. **对赌驱动价值创造**：以可换股贷款+业绩对赌方式介入困境企业（微创医疗），设定三年扭亏路径
4. **硬件+AI融合**：医疗器械投资方向融入"硬件+AI"解决方案

### 4.3 战略转型（2024-2026）

| 维度 | 变化 |
|------|------|
| **区域战略** | 从"重仓中国"→淡化"中国"标签→全球化（日本、印度、中东、欧洲） |
| **资产类别** | 从单一PE→PE+VC+二级市场+不动产+并购全覆盖 |
| **基金策略** | 六期基金Buyout $7-8B + Growth $1-1.5B，分工更精细 |
| **GP出资** | 六期基金GP承诺$1B（~14.2%），历史最高比例 |
| **消费赛道** | 引入百事前CEO陈文渊，消费与产业运营深度融合 |
| **AI重仓** | 2024-2025年AI/具身智能成为最大新增重注方向 |

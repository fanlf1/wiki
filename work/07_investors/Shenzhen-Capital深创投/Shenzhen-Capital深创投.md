# Shenzhen Capital深创投

## 一句话定位
> 中国本土最大创投机构，累计投资1800+企业、托举274+家企业上市，是硬科技IPO退出之王。国资背景使其投资方向反映国家政策导向。

## 与我们的关系
- **关注优先级**：高（A级 产业验证）
- **核心关注点**：深创投是硬科技领域最活跃的本土VC，IPO退出数量行业第一；设有专门的健康产业基金；近年开始布局脑机接口等前沿赛道
- **信号价值**：投谁=政策导向+技术判断。深创投的投资方向反映国家战略对技术的判断

## 四、投资逻辑/论文

### 核心哲学

| 维度 | 内容 |
|------|------|
| **使命** | "发现并成就伟大企业" |
| **核心策略** | **"国家需要什么，深创投就投什么"** |
| **投资方针** | 投早、投小、投长期、投硬科技 |
| **方法论** | 研投一体，把关键核心技术清单（"卡脖子"清单）转化为重点投资方向 |
| **技术洞察** | "硬科技项目藏在产业链深处，创投需融汇产学研资" |

### 投资偏好

1. **硬科技为核心**（90%+）：前董事长靳海涛最早将硬科技做成主赛道，影响了整个中国创投行业
2. **全生命周期覆盖**：从种子/天使到Pre-IPO全阶段
3. **研投一体**：主动研究驱动，绘制重点产业投资图谱，而非被动看项目
4. **政策导向**：国资背景使投资方向高度贴合国家战略
5. **产业深耕**：七大战略新兴产业（信息科技、智能制造、生物医药与健康、新能源/新材料、半导体、消费、企业服务）

### 2024年投资金额分布（245场投资决策会）

| 领域 | 占比 |
|------|------|
| 新材料与新能源 | 60% |
| 新一代信息技术 | 25% |
| 高端装备制造 | 8% |
| 生物科技与健康 | 7% |

> 2024年逆势完成投资金额**137亿元**，新设基金14只。

### 医疗健康投资逻辑

- 设立专门的健康产业基金（红土医疗基金），由专业医学背景团队管理
- 重点领域：IVD与生命科学、创新药、医疗器械、智慧医疗
- 荣获"年度IVD与生命科学最佳投资机构"（医疗健康投资卓悦榜）
- 产业链资源嫁接优势：利用庞大被投企业网络做上下游协同（如元化智能手术机器人与骨科企业合作做临床试验）

### 前沿布局逻辑

- 设立**早期投资部**，举办**种子训练营**
- 五大前沿方向：**人形机器人、脑机接口、通用人工智能、量子计算、可控核聚变**
- 种子训练营无地域限制、无次数限制，持续筛选前沿项目
- 早期项目占比接近50%，持续加大早期投资力度

### 战略判断

1. **深创投是"政策风向标"**：其投资方向反映国家层面对技术路线的判断
2. **医疗健康是重点赛道**：红土医疗基金20亿+规模，由专业医学博士团队管理
3. **脑机接口值得关注**：深创投已明确将其列为种子训练营五大方向，说明国资层面看好
4. **CGM/戒指/长寿暂无信号**：这些细分领域不是深创投当前重点
5. **退出能力是核心竞争力**：274+家IPO退出记录，对创业者有强吸引力

### 融资接触建议

1. **首选接触路径**：红土医疗基金 → 周伊团队
2. **项目定位**：医疗级可穿戴健康监测设备（国产替代 + 卡脖子技术突破）
3. **阶段匹配**：A轮/B轮阶段，符合深创投"投早投小"策略
4. **差异化卖点**：
   - 医疗级传感器（区别于消费级可穿戴）
   - 连续血糖监测/血压监测等核心功能
   - 与深圳产业链配套（硬件+制造优势）
5. **关联被投企业**：联新医疗（智慧病房）、华声医疗（超声设备）、鑫精诚（传感器）——可做产业链协同叙事

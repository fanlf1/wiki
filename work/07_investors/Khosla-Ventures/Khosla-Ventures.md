# Khosla Ventures

## 一句话定位
> Vinod Khosla创立的深科技VC，以"敢于投不可能"著称。投了OpenAI、DoorDash、Square等超级赢家，在生物传感+健康穿戴方向有独特品味（AliveCor、SiPhox、Synchron）。健康医疗是其八大投资赛道之一，已退出Guardant Health、Oscar Health。

## 基本信息

| 项目 | 内容 |
|------|------|
| 英文名 | Khosla Ventures |
| 类型 | VC |
| 总部 | Menlo Park, California（Sand Hill Road 2128号） |
| 创始人 | Vinod Khosla（Sun Microsystems联合创始人，前Kleiner Perkins合伙人） |
| 成立时间 | 2004年（前两期基金用个人资本，未向机构LP开放） |
| 阶段偏好 | Seed到成长期，偏好大胆的技术赌注 |
| 重点赛道 | AI、深科技、可持续、数字健康、MedTech、金融科技、前沿技术（共8大类） |
| 官网 | https://khoslaventures.com/ |
| 总AUM | ~$18B（2024中） |

### 基金历史

| 基金/时间 | 规模 | 备注 |
|-----------|------|------|
| Fund I（2004） | ~$250M | 创始基金，个人资本 |
| Fund III + Seed（2009.9） | $1B + $300M Seed | 首次大规模募资，$300M Seed基金专投高风险科学实验 |
| 2021.10 | $1.4B | $400M seed + $1B late-stage |
| Opportunity Fund I（2022.1） | $557M | 首只机会基金 |
| Fund VIII + Growth + Seed（2023.11） | $3.1B | $1.6B旗舰 + $1B成长期 + $500M种子期；当年VC最大募资 |
| OpenAI SPV（2024.10） | $405M | 专项投资OpenAI |
| 2024-2025新一轮 | ~$3.5B（目标） | 旗舰+种子+成长三只基金 |

**趋势**：基金规模持续扩大，从$5B AUM（2017）到$18B AUM（2024），增长约3.6倍。市场高度认可。

## 与我们的关系
- **关系类型**：行业风向标 + 潜在融资方
- **关注优先级**：高
- **核心关注点**：Khosla偏好"用技术解决不可能问题"——无创血糖正是这类方向。SiPhox的硅光+居家检测思路与我们的多标志物光谱检测有底层逻辑相似

## 投资哲学

- **"Innovation = doing the impossible"**：Khosla以大胆押注前沿技术著称
- **"Be obstinate about your vision, flexible about your tactics"**：对愿景固执，对战术灵活
- 不追风口，投"大多数人不信但可能改变世界"的方向
- 与Cleveland Clinic合作（2025.10起），推动新兴医疗技术规模化——Khosla portfolio公司可在Cleveland Clinic真实临床场景中测试
- **"99%的时间你会得到虚伪的礼貌，要敢于反对"**——强调brutal honesty文化
- 2023年获"Founder's Choice VC"第一名（200+VC中由创始人投票选出）

## 值得关注的信号

- Khosla如果投无创血糖/光谱传感方向，说明这个方向从"不可能"变成"有可能"
- SiPhox的硅光技术路线与我们[[../../04_research/多标志物-光谱检测/|多标志物光谱检测]]课题有参考价值
- Khosla的风格适合早期深科技项目，如果我们做无创血糖戒指，属于其投资偏好
- **Alex Morgan**（Partner）和**Jun Jeon**（Partner）是健康医疗方向核心决策者，应跟踪其公开言论
- **Cleveland Clinic合作**（2025.10）为portfolio公司提供临床测试场景，如果我们成为portfolio公司也可受益
- Synchron的BCI投资说明Khosla对"植入式生物传感"有偏好，与我们的穿戴式路线有差异但底层逻辑一致
- Ultima Genomics的$80基因组测序目标说明Khosla对"把昂贵医疗技术白菜化"有强兴趣——无创血糖也有类似叙事
- Khosla 2023年获"Founder's Choice VC"第一名，说明创始人社区认可其风格，融资谈判可能更顺畅

# Microsoft微软投资

## 一句话定位
> 微软通过M12基金（企业风投）和战略直投两大路径布局前沿科技；M12专注企业级早期投资，战略投资层面对OpenAI等平台级公司重仓$13B+，并在AI+医疗/BCI方向通过Azure合作生态扩展影响力。

## 与我们的关系
- **关注优先级**：B级（补充信号）— 健康穿戴/生物传感/数字健康方向有布局，但非核心聚焦赛道
- **关联品牌**：[[Figure AI]]（Robot赛道，投资方）、INBRAIN Neuroelectronics（BCI合作伙伴）
- **关键信号**：微软与INBRAIN的BCI合作、Azure AI+医疗健康生态、M12在数字健康的多笔投资

## 四、投资逻辑/论文

### 核心投资哲学

1. **财务回报优先**：M12明确声明是"return-driven fund"，不以战略收购为目的，已取消"优先购买权"（ROF）等限制性条款
2. **与微软战略协同**：投资方向与微软的云（Azure）、AI、企业软件生态紧密对齐
3. **微软"肌肉"加持**：被投公司可获得微软全球销售渠道、Azure信用额度、技术资源支持
4. **干净条款**：几乎所有termsheet都是clean的，无限制性条款

### 投资偏好

| 维度 | 偏好 |
|------|------|
| 阶段 | 早期为主（A轮、B轮），近年增设种子子基金 |
| 领域 | 企业级技术：AI/ML、云基础设施、网络安全、开发者工具、商业应用 |
| 单笔 | $5M–$20M（常规）；$500K–$2M（种子） |
| 地域 | 全球（北美、欧洲、以色列、印度） |
| 特殊关注 | 多元化创始人投资（设Female Founders竞赛，已部署$10M+） |

### 投资分类（M12官网定义）

- AI Apps / AI Cloud Infrastructure / AI Data Ops & Models
- Cloud Infrastructure / Cybersecurity / Deep Tech and AI
- Developer Tools / Enterprise Applications / Vertical SaaS
- Web3 and Gaming / Security for AI

### 对健康/生物赛道的定位

> "I'm looking forward to what the next five years will bring with continued discovery, innovation and growth — **particularly in health care and life sciences**, cybersecurity and productivity."
> — Michelle Gonzalez, M12 Global Head

微软在健康赛道的策略更偏向**AI+数据层**（如Innovaccer的数据平台、1910 Genetics的AI制药）而非**硬件层**（不做传感器/可穿戴投资）。BCI方向通过与INBRAIN的合作探索"AI+云+神经接口"模式。

## 七、对我们赛道的信号评估

| 赛道 | 微软/M12动作 | 信号强度 | 解读 |
|------|-------------|----------|------|
| **CGM/血糖** | 无直接投资；Azure间接支持糖尿病数据平台 | 弱 | 微软不做硬件投资，CGM非其视野 |
| **智能戒指** | 无投资 | 无 | 完全不在微软投资视野内 |
| **BCI/脑机** | INBRAIN战略合作（Azure AI+石墨烯神经接口） | 中 | 合作模式而非投资；"AI+BCI"方向有长期价值 |
| **可穿戴设备** | 无硬件投资；曾自研Band手环已停产 | 弱 | 微软已退出消费级可穿戴硬件 |
| **生物传感** | 无硬件投资；1910 Genetics涉及生物自动化 | 弱 | 偏AI/数据层，不碰传感器硬件 |
| **数字健康** | Innovaccer、Andor Health、RAAPID、iLoF | 中 | M12在数字健康有4+笔投资，是最相关的赛道 |
| **长寿科技** | 无投资 | 无 | 不在微软视野内 |

### 总结判断

微软/M12在我们的核心赛道（CGM、智能戒指、生物传感、可穿戴）上**几乎没有直接投资**，原因是：
1. M12聚焦企业级软件/AI，不做硬件投资
2. 微软自身体内已有健康AI方向（Azure for Healthcare），但通过合作/平台模式而非投资
3. 唯一的直接信号是**INBRAIN BCI合作**和**数字健康方向的多笔投资**

**建议关注点**：微软的Azure AI+医疗生态可能是被投/合作伙伴的技术底座；INBRAIN合作模式值得跟踪，可能预示微软在BCI领域的进一步动作。

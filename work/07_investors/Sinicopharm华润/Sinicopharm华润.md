# CR Capital华润资本

## 一句话定位
> 华润集团旗下私募股权投资管理平台（集团产业基金唯一运营管理平台），累计管理规模超1200亿元，围绕华润大健康生态进行战略投资与产业协同。

## 投资方向与策略

### 核心策略："基金+产业"双轮驱动

华润资本的投资逻辑核心是**产业协同**：
- 大多数基金由"华润资本 + 产业公司 + 市场化资金"共同出资
- 华润集团出资比例不超过基金规模的25%
- 所投项目需与华润集团业务有协同作用

### 重点投资赛道

华润医药产业基金聚焦：
1. **化学创新药**
2. **生物药**
3. **高端医疗器械**
4. **中药滋补品类**
5. **合成生物学**

其他方向：新能源/环保、消费品/零售、城市更新、科技创新

## 投资逻辑

1. **产业协同优先**：张宗公开表述"投资项目需与华润集团业务有协同作用"
2. **基金+产业双轮驱动**：基金配合华润各业务单元（华润医药、华润三九等）的战略投资
3. **"先投后并"模式**：基金先期挖掘投资优质标的，再帮助产业落地（如天东制药案例）
4. **央企资源赋能**：依托华润集团超2000家企业资源池进行产业协同
5. **战略新兴产业直投**：2023年调整为集团直属机构后，强化战新产业直投职能

## 最新动态

- **2025年底**：华润双鹤产业基金、华润医药成都产业基金完成备案
- **2025年7月**：华润医药产业投资基金二期设立（目标规模10亿元）
- **2025年初**：华润医药商业完成62.60亿元引战混改，为近三年医疗医药行业最大单笔私募融资
- **2025年4月**：孵化企业亿杉医疗入选香港特区政府重点企业伙伴
- **2024年**：华润双鹤收购华润紫竹100%股权，进入女性健康领域
- **秦锋职务变动**：秦锋已调任华润资产经营有限公司总经理，华润资本CEO可能已更替


# 联想创投（Lenovo Capital）

## 一句话定位
> 联想集团旗下全球科技产业基金，聚焦IoT+边缘计算+AI，在生物传感器和脑机接口有精选布局。

| 项目 | 内容 |
|------|------|
| 英文名 | Lenovo Capital & Incubator Group (LCIG) |
| 中文名 | 联想创投集团 |
| 类型 | CVC（企业战投） |
| 总部 | 北京 |
| 管理规模 | 超216亿人民币 |
| 阶段偏好 | 全阶段（早期+成长期） |
| 成立 | 2016年 |
| 重点赛道 | IoT、边缘计算、云、大数据、AI、AI+行业、消费升级 |
| 被投企业数 | 300+（含25家IPO，50+独角兽） |

## 与我们的关系
- **关系类型**：行业风向标 + 潜在合作伙伴
- **关注优先级**：中
- **核心关注点**：联想在传感器（钛方科技）和脑机接口（智冉医疗）的投资，与我们BCI耳机和血糖戒指的供应链和生态有关联

---

## 投资哲学

- **"科技产业CVC"**：不做纯财务投资，强调被投企业与联想业务的协同
- 2025年主题"硅基进化，万物共生"——围绕硅基智能产业链（底层算力→创新架构→上层应用）
- 已推动近30家被投企业与联想集团展开实质性合作

### Portfolio细节补全

#### bio-sensor / 钛方科技
- **投资时间线**：联想创投2019年起投资并推动与联想业务合作
- **生态协同**：2019年起产品陆续应用于联想笔记本电脑HapticPad 2.0；联合定义三合一触控芯片，从二级供应商升级为一级模组供应商
- **技术壁垒**：弹性波传感器、芯片与算法全栈技术，百余项专利
- **应用场景**：汽车电子、消费电子（笔记本触控板），未来可能扩展到生物力学测量

#### bci / 智冉医疗
- **轮次/金额**：A轮超3亿元（2024-2025）
- **技术方向**：新一代侵入式柔性脑机接口平台
- **与我们关系**：非消费级方向，但说明联想在关注BCI赛道，可作为技术路线对标

#### 算力芯片组合（联想创投核心赛道）
- **投资逻辑**：围绕AI四个要素——能源、算力、算法、数据科学布局
- **"二八"原则**：80%聚焦核心科技，20%投资未来最有想象力的领域
- **研究驱动**：每年12月至次年2月进行三个月深度行研，期间一个项目都不投
- **算力矩阵**：寒武纪（AI专用芯片）→ 海光（x86兼容）→ 摩尔线程（全功能GPU）→ 沐曦（数据中心GPU）→ 芯驰科技（车载计算，投后估值超140亿）→ 此芯科技（端侧AI CPU，45TOPS）

#### 具身智能/机器人
- **投资规模**：累计投资超30家相关企业
- **代表项目**：云迹科技（已上市）、它石智航（1.2亿美元天使轮，创中国具身智能行业天使轮最大融资纪录）
- **趋势判断**：自动驾驶人才和技术"外溢"到具身智能；2025年是"具身进入工厂的元年"

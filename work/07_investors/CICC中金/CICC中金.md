# CICC中金公司

## 一句话定位
> 中国顶尖投资银行，中金资本管理私募股权规模突破6800亿元，是国内最大母基金+直投平台之一，2023年医疗健康投资活跃度TOP3，通过脑科学基金、阿斯利康联合基金等专项布局医疗科技。

## 与我们的关系
- **关注优先级**：高
- **核心关注点**：中金是医疗健康和硬科技IPO的核心保荐机构（2020年来医疗IPO 63单），对我们未来融资和上市路径有直接影响；脑科学基金已投傲意科技（肌电/脑电传感），与我们的生物传感方向有交集
- **证据力等级**：C级（背景参考）-- 管理规模巨大但赛道专注度不够集中，以母基金/PE为主而非早期VC

## 投资逻辑/论文

### 核心投资理念

1. **"研究驱动投资"** -- 中金资本的核心投资理念，依托中金公司强大的研究能力
2. **"资本支持与赋能服务双轮驱动"** -- 构建股权投资生态，不仅提供资金还提供综合金融服务
3. **"投早、投小、投长期、投硬科技"** -- 契合国家政策导向，通过母基金+直投覆盖种子到PE全阶段

### 投资策略特点

- **母基金+直投双轮驱动**：通过参股设立母基金/子基金吸引撬动社会资本，同时进行精准直投
- **全链条覆盖**：覆盖"募投管退"全链条，从种子/天使到PE/并购
- **产业方合作**：与阿斯利康等产业方联合设立专项基金，整合产业资源和资本优势
- **政府引导基金**：管理大量国家级/地方级政府引导基金，政策导向性强
- **综合金融服务**：为被投企业提供投融资、财务顾问、IPO承销等全方位服务

### 投资偏好

| 维度 | 偏好 |
|------|------|
| 阶段 | 偏中后期（PE阶段为主），通过母基金间接覆盖早期 |
| 行业 | 硬科技、医疗健康、先进制造、新能源 |
| 地域 | 全国布局（北京总部+厦门/深圳/苏州等地方基金） |
| 合作模式 | 政府引导基金+产业方联合基金+市场化母基金 |

### 综合判断

1. **优势**：
   - 国内最大PE/母基金平台之一，资金实力雄厚
   - 医疗健康IPO保荐经验丰富（63单），未来上市路径可借助
   - 脑科学基金专项布局与我们的生物传感方向有交集
   - 与阿斯利康等产业方有深度合作经验

2. **风险**：
   - 2025年核心管理层大面积被调查，投资决策可能受影响
   - 以母基金/PE为主，与我们的早期项目阶段不太匹配
   - 赛道专注度不够集中，除脑科学基金外无专项聚焦我们赛道的基金
   - CGM/血糖监测、智能戒指、长寿科技等赛道尚未发现投资布局

3. **对我们的意义**：
   - **未来融资**：中后期（B轮以后）可考虑接触中金启元/启德基金
   - **上市路径**：中金是医疗健康IPO顶级保荐人，上市时优先考虑
   - **赛道信号**：脑科学基金的傲意科技投资说明中金看好非侵入式神经传感方向，可视为我们生物传感方向的间接验证

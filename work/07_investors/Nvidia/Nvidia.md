# Nvidia英伟达投资（NVentures）

## 一句话定位
> GPU/AI芯片巨头NVIDIA旗下企业风险投资部门NVentures，从公司资产负债表出资，专注投资AI驱动的技术愿景公司，2025年完成约67笔VC交易（2024年54笔），已成为全球最活跃的CVC之一；医疗健康/AI药物发现是其重点布局方向。

## 基本信息

| 项目 | 内容 |
|------|------|
| 英文名 | NVentures（NVIDIA Venture Capital） |
| 类型 | CVC（企业风险投资） |
| 总部 | 美国加州Santa Clara |
| 母公司 | NVIDIA Corporation（NASDAQ: NVDA，市值~$4.6万亿） |
| 成立时间 | 2021年（2023年正式对外公布） |
| 资金来源 | NVIDIA公司资产负债表（非传统基金模式） |
| 单笔投资规模 | $20M–$100M+ |
| 投资阶段 | Seed、Series A及以后各阶段 |
| 负责人 | Mohamed (Sid) Siddeek（企业副总裁兼NVentures负责人） |
| 官网 | https://www.nventures.ai/ |

## 与我们的关系
- **关系类型**：行业风向标 + 潜在生态合作方
- **关注优先级**：高
- **核心关注点**：NVentures在AI医疗（Hippocratic AI）、AI药物发现（Genesis、Charm、EvolutionaryScale）、医疗机器人（Neocis）等领域积极布局，与"AI+生物传感""AI+数字健康"赛道有交集；NVIDIA的GPU/AI基础设施是我们产品AI能力的底层依赖
- **关联品牌**：[[Figure AI]]（Robot赛道，NVIDIA战略投资）、[[Hippocratic AI]]（医疗LLM，NVentures投资）

## 1. 基金规模与历史

### 投资架构：双轨制

NVIDIA的投资活动通过两个团队并行运作：

| 维度 | NVentures（风险投资） | Corporate Development（企业发展） |
|------|----------------------|-------------------------------|
| 负责人 | Sid Siddeek | Vishal Bhagwati |
| 定位 | 早期/成长期投资，追求财务回报 | 战略性大规模投资/收购，增强NVIDIA平台 |
| 单笔规模 | 数百万至数千万美元 | 数亿至数十亿美元 |
| 代表交易 | Genesis Therapeutics, Charm Therapeutics, Hippocratic AI | Figure AI ($675M→$1B+), Mistral AI ($2B), xAI ($6B) |
| 回报导向 | 财务回报 + 战略协同 | 纯战略导向 |

### 关键数据

| 指标 | 数值 |
|------|------|
| 2022年交易数 | 1笔 |
| 2023年交易数 | ~12笔 |
| 2024年交易数 | 54笔（含企业发展和NVentures） |
| 2025年交易数 | ~67笔（NVentures单独30笔） |
| 2026年至今（YTD） | 19笔 |
| 独角兽被投 | 18家 |
| 已知退出 | 1个IPO + 1个收购（Tracxn数据） |
| 投资总额（2025年NVIDIA整体） | $40B+ |

### 资金来源

- NVentures从NVIDIA公司资产负债表直接出资，**不设独立基金**
- 不存在传统意义上的"基金规模"或"募资周期"
- 资金充裕度直接与NVIDIA的盈利和市值挂钩（2025财年营收$609亿，净利润$1,673亿）

## 4. 投资逻辑/论文

### 核心论点

> **"AI将触及每一个行业"**

NVentures的投资哲学核心：

1. **AI + X**：寻找AI能够颠覆性改造的传统行业
2. **生态增强**：投资能够使用和推广NVIDIA技术的公司
3. **回报导向**：不同于纯战略CVC，NVentures追求财务回报

### 投资评估三大维度

| 维度 | 核心问题 |
|------|----------|
| **团队（Team）** | 团队实力、背景、深度、谦逊品质 |
| **技术平台（Technology）** | 技术的健壮性和5-10年的持久性；"kick the tires"深度技术验证 |
| **市场潜力（Market）** | 大型被颠覆市场 vs. "零到十亿"全新市场（zero-billion dollar market） |

### 行业布局策略

按Sid Siddeek的表述，NVentures按以下优先级逐步扩展：

1. **AI基础设施/核心AI**（最先布局）
2. **医疗健康/生物技术**（第二波重点——"tech bio"）
3. **制造业**（已开始——Seurat、Bright Machines）
4. **金融服务**（感兴趣但尚未开始）
5. **其他行业**（按顺序探索）

### 医疗健康领域的具体逻辑

- **药物发现**是重点：开发一个新药需~$10亿，AI可以在湿实验前大幅提升成功率（in silico筛选）
- **"Tech Bio"模式**：计算 + 生物实验结合，而非纯计算替代
- NVIDIA的差异化优势：GPU计算能力 + Clara医疗计算平台 + BioNeMo药物研发平台

### 与传统CVC的差异

| 维度 | NVentures | 典型CVC |
|------|-----------|---------|
| 回报导向 | 财务回报 + 战略协同 | 纯战略导向 |
| 投资阶段 | Seed到Growth全覆盖 | 通常偏向后期 |
| 技术验证 | NVIDIA内部技术团队深度参与（"kick the tires"） | 较少深度技术尽调 |
| 投后支持 | NVIDIA全套技术和GTM资源赋能 | 品牌/渠道资源 |

## 值得关注的信号

1. **NVentures是AI医疗/TechBio领域最活跃的CVC**——如果我们在AI+生物传感方向需要战略投资，NVentures值得关注
2. **投资逻辑偏"AI软件+计算"而非硬件**——NVentures不太可能直接投资可穿戴硬件，但如果我们的AI算法层足够强，有可能获得关注
3. **"投资→acquihire"路径**：Enfabrica案例说明NVentures投资可能是NVIDIA后续收购的前奏
4. **NVIDIA的差异化价值主张**：不仅给钱，还给GPU计算资源+NVIDIA技术生态+GTM支持——对于依赖AI的公司极具吸引力
5. **Sid Siddeek提到金融科技是下一个探索方向**——说明NVentures在持续扩展赛道

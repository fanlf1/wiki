# 美团龙珠资本（Dragonball Capital）

## 一句话定位
> 美团旗下产业投资基金，管理规模超130亿元，从消费赛道起家，近年大举进军AI+医疗健康、机器人、生物传感等硬科技领域。

## 基本信息

| 项目 | 内容 |
|------|------|
| 英文名 | Dragonball Capital（Long-Z Capital） |
| 中文名 | 美团龙珠资本 |
| 管理主体 | 宁波梅山保税港区美兴私募基金管理有限公司 |
| 类型 | CVC（产业投资基金） |
| 总部 | 北京 |
| 母公司 | 美团（HKSE: 3690） |
| 成立时间 | 2017年 |
| 管理规模 | 超130亿元人民币（3期人民币基金 + 1期美元基金 + 成都专项基金18亿，共11只在管基金） |
| 阶段偏好 | A轮为主，覆盖B轮及以后 |
| 重点赛道 | 消费科技（传统优势） → AI/大模型、机器人、医疗健康、生物传感、手术机器人（近年拓展） |

### 基金沿革

| 基金 | 规模 | 时间 | 备注 |
|------|------|------|------|
| 一期人民币基金 | 20亿元 | 2017年11月首期交割，2018年7月最终交割 | 出资人：美团、腾讯等；专注消费C轮前项目 |
| 二期人民币基金 | 约50亿元 | 2021年前后 | 腾讯再次作为LP出资 |
| 三期人民币基金 | 未公开 | 2023-2024年 | 方向拓展至AI、硬科技 |
| 美元基金（一期） | 未公开 | — | 面向全球募资 |
| 成都专项基金 | 18亿元 | — | 与成都政府引导基金合作 |

### LP构成

- 美团（母公司）
- 腾讯（多期基金LP）
- 政府引导基金（成都等）
- 全球主权基金
- 养老基金
- 知名母基金
- 消费行业龙头企业

## 与我们的关系
- **关系类型**：行业风向标 + 潜在合作伙伴
- **关注优先级**：中（从C级上调 -- 近期医疗健康/生物传感布局密集）
- **核心关注点**：
  - 2026年3月投资深纳普思（AI+生物传感可穿戴），与我们的生物传感/可穿戴赛道高度重合
  - 连续领投心航路医学（6亿B轮）、追投康诺思腾（8亿轮），显示对医疗器械/数字健康的坚定布局
  - 从纯消费基金转型为"消费+科技+医疗"综合基金，投资逻辑演变值得关注
- **关联品牌**：[[宇树科技]]、深纳普思、心航路医学、康诺思腾

## 投资哲学

**核心方法论：看准大行业机会里的头部公司，果断出手并长期持有（集中投资策略）。**

- 美团龙珠以"集中投资"著称，一期基金20亿元未分散投资大量项目，而是精选重仓
- 从消费科技起家，近年明确提出向"AI+硬科技+医疗健康"拓展
- 美团CFO陈少晖曾表示：龙珠的投资围绕美团的战略延伸，寻找能够与美团生态产生协同的机会
- 在医疗健康领域的投资逻辑：AI赋能的生物传感器是连接物理世界与数字健康的关键载体，与美团布局硬科技、拓展科技健康服务的战略高度契合

## 值得关注的信号

1. **2026年3月 -- 投资深纳普思**：这是美团龙珠首次直接投资AI+生物传感可穿戴设备公司，标志着其医疗健康布局从手术机器人延伸至医疗级可穿戴赛道。深纳普思专注于AI赋能生物传感器和慢病管理平台，与CGM/血糖监测/数字健康赛道高度相关。

2. **2025年5月 -- 联合领投心航路医学6亿B轮**：当月医疗健康领域金额最高的投资事件，显示龙珠在医疗器械领域的持续加码。

3. **投资方向转型明显**：从纯消费（茶饮、咖啡）→ 消费+科技（AI大模型、机器人）→ 消费+科技+医疗健康（生物传感、手术机器人、心脏电生理）。三期基金明确将AI和硬科技作为重点方向。

4. **与礼来亚洲基金（LAV）形成联合投资默契**：心航路医学、康诺思腾两案均与LAV联合投资，暗示龙珠正在建立医疗健康领域的专业投资网络。

5. **成都专项基金18亿元**：与地方政府合作设立专项基金，可能意味着在医疗器械/传感器等地方优势产业上会有更多布局。

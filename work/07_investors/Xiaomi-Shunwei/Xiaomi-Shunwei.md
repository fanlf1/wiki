# 小米战投 & 顺为资本（Xiaomi & Shunwei）

## 一句话定位
> 小米战投+顺为资本是"硬件生态链"投资模式的定义者，在智能硬件和可穿戴领域有最密集的投资布局。

## 基本信息

| 项目 | 内容 |
|------|------|
| 英文名 | Xiaomi Strategic Investment / Shunwei Capital |
| 中文名 | 小米战略投资部 / 顺为资本 |
| 类型 | CVC（小米战投）+ 独立VC（顺为） |
| 总部 | 北京 |
| 管理规模 | 顺为：$2.5B+（美元基金）；小米战投：未公开 |
| 阶段偏好 | 早期到成长期 |
| 关系 | 雷军同时创立小米和顺为，两者常联合投资 |
| 重点赛道 | 智能硬件、IoT、消费电子、AI、机器人、电动汽车 |

## 与我们的关系
- **关系类型**：竞品生态构建者 + 行业风向标
- **关注优先级**：高
- **核心关注点**：小米在智能穿戴领域的生态链模式（华米/Amazfit、小米手环）已定义了"性价比健康穿戴"品类，小米如果做血糖戒指将是最大竞争威胁

---

## 投资哲学

- **"铁人三项"**：硬件+互联网+新零售，用投资构建生态链
- 顺为侧重财务回报+战略协同，小米战投侧重供应链整合+生态扩展
- 两者经常**联合投资**同一项目（如华米、乐骑智能、迈塔兰斯）
- 健康科技理念："**云（健康云服务）+端（可穿戴终端）+芯（芯片）**"

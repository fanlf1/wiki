## 技术评估

### 核心技术

**GO-1大模型（ViLLA架构）**
- 2025年3月发布，全球首个通用具身基座模型
- ViLLA = Vision-Language-Latent-Action，将VLA进化到新阶段
- 可利用人类视频学习，小样本快速泛化
- 已开源

**PowerFlow关节电机**
- 准直驱关节方案，低齿槽转矩设计
- 搭配10速比以内高力矩透明度行星减速器
- 峰值扭矩 >350Nm，重量仅1.6kg
- 共钜同轴双编码器 + 一体液冷循环散热
- 自研矢量控制驱动器
- 径向磁通外转子方案（未来有轴向磁通版本）

**SkillHand灵巧手**
- A2: 12主动+5被动自由度
- A3: **19主动自由度**（从12升级）
- 基于视觉的指尖传感器，可分辨颜色、形状、材质
- 视觉闭环降低对电机精度需求

**"一体三智"架构 + EI-Brain具身智脑**

| 模块 | 含义 |
|------|------|
| 一体（本体） | 机器人物理载体 |
| 运动智能 | 四肢协调、双足行走、动态运动 |
| 交互智能 | 听说看理解、自然语言、多模态感知 |
| 作业智能 | 工业操作、精密装配、家务任务 |

EI-Brain四层架构：云端超脑 → 大脑（任务调度） → 小脑（运动控制） → 脑干（电机底层）

## 优势与风险

**优势**
1. GO-1 ViLLA模型是通用具身智能的先发者
2. 全栈开源策略构建开发者生态（AgiBot World获IROS最佳论文）
3. 交互体验被评价为"优于宇树"
4. 稚晖君个人IP和华为天才少年的品牌效应
5. 2年150亿估值的融资效率极高

**风险 / 劣势**
1. 运动控制能力弱于宇树（行业共识）
2. 成立时间短（2023年），工程成熟度待验证
3. 开源策略的商业模式尚未清晰（如何从生态中盈利）
4. Pre-IPO阶段，面临上市前业绩压力

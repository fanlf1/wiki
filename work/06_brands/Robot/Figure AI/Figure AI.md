## 技术评估

### 核心技术：Helix VLA模型

Helix是Figure AI的核心技术壁垒，是一个Vision-Language-Action (VLA)基础模型：

| 特性 | 详情 |
|------|------|
| 控制范围 | 全上肢（手腕、手指、躯干、头部） |
| 控制频率 | 200 Hz |
| 架构 | 双系统设计（推理+快速动作） |
| 零样本能力 | 无需训练即可识别和操作新物体 |
| 输入方式 | 自然语言指令 + 视觉 |

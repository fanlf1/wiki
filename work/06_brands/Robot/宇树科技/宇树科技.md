## 技术评估

### 核心技术
- **运动控制**：源自四足机器人的动态平衡和步态控制，迁移至人形机器人；格斗算法突破，高难度动态平衡和敏捷运动
- **自研执行器全栈能力**：永磁同步电机(PMSM) + 减速器 + 编码器 + 传感器全部自研
- **性价比工程**：G1以¥9.9万实现23DoF+120Nm膝关节，成本控制能力极强
- **设计哲学**：王兴兴提出"越智能的机器人，应该越是克制"——强调安全性与可控性

### 技术壁垒
1. 四足机器人全球市占率>60%的规模效应和供应链积累
2. 自研执行器+减速器+编码器的全栈硬件能力
3. 运动控制算法在四足->人形之间的技术迁移
4. 从设计到量产的极致成本压缩能力（9年估值增长1270倍的核心驱动）

## 优势与风险

**优势**
1. 全球人形机器人出货量第一，先发优势明显
2. 四足机器人69.75%市占率的供应链和制造经验
3. G1 ¥9.9万 / R1 $5,900 的极致性价比碾压竞品
4. 60%毛利率，2025年净利润6亿，已实现盈利
5. 产品矩阵完整：R1(消费)-G1(开发)-H1(工业)，覆盖全场景
6. 科创板IPO推进，资本化加速

**风险 / 劣势**
1. G1尺寸小（127cm），与全尺寸竞品不在同一赛道
2. AI/大模型能力弱于Figure AI（Helix）和Tesla（FSD迁移）
3. IPO后面临资本市场短期业绩压力
4. 四足机器人市场增长见顶，需人形机器人接力
5. 智元机器人出货量紧随其后，竞争白热化

# Senseonics Eversense

## 一句话定位
> 全球唯一获批的植入式CGM，Eversense E3传感器可在皮下工作180天，开创长期植入监测方向。

## 与我们的关系
- **对标关系**：技术参考（植入式长期监测的思路与无创长期监测有共鸣）
- **关注优先级**：中
- **核心关注点**：证明了用户愿意接受"一次植入/佩戴，长期使用"的模式，这对无创方案的商业模式有参考价值

### Eversense E3（核心产品）

**核心参数**
| 参数 | 规格 |
|------|------|
| 传感器寿命 | 180天（皮下植入） |
| MARD | 8.5% |
| 校准 | 每天1-2次指尖血校准 |
| 测量频率 | 每分钟 |
| 数据传输 | 蓝牙实时传输（需佩戴贴片式发射器） |
| 植入方式 | 门诊小手术，皮下植入3-5mm |
| 监管认证 | FDA / CE |

**技术路线**
- 荧光传感：皮下植入荧光传感器 → 体液葡萄糖与荧光聚合物结合 → 外部贴片发射器光学读取荧光信号变化 → 算法转换血糖值。与传统电化学路线完全不同

## 技术评估

### 核心技术
- **荧光葡萄糖传感**：独有技术路线，传感器不消耗酶，理论寿命远超电化学
- **光学遥测**：通过皮肤读取体内荧光信号，是"跨皮肤无创读取"的先驱

### 技术壁垒
- 唯一获批的植入式CGM，监管壁垒高
- 荧光传感专利布局完整
- 180天寿命的化学稳定性技术

### 关键局限
- 需门诊小手术植入/取出
- 仍需每天校准1-2次（不及Abbott/Dexcom的免校准）
- 需佩戴外部贴片发射器

## 优势与风险

**优势**
1. 180天植入寿命，更换频率远低于所有竞品
2. 每分钟测量频率，数据密度最高
3. 荧光技术路线独特，长期技术潜力大
4. 植入体验→"装了就忘"，用户依从性可能更好

**风险 / 劣势**
1. 需门诊手术植入/取出，准入门槛高
2. 仍需每天校准，不如竞品便捷
3. 市场份额极小，商业化能力待验证
4. 外部贴片发射器需每24小时充电

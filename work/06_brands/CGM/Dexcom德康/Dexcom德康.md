# Dexcom 德康

## 一句话定位
> 全球CGM技术精度领先者，以MARD 8.0%的G7定义行业精度标杆。

## 与我们的关系
- **对标关系**：直接竞品 + 技术标杆
- **关注优先级**：高
- **核心关注点**：MARD 8.0%是无创方案需要对标的核心精度基准；其实时监测体验（每5分钟）是用户期望的参照

### Dexcom G7（旗舰产品）

**核心参数**
| 参数 | 规格 |
|------|------|
| 传感器寿命 | 15天 |
| MARD | 8.0% |
| 校准 | 无需校准（工厂预校准） |
| 预热时间 | 30分钟 |
| 测量频率 | 每5分钟 |
| 数据传输 | 蓝牙5实时传输 |
| 防水等级 | IP28 |
| 监管认证 | FDA / CE |

**技术路线**
- 电化学原理：皮下植入传感器 → 葡萄糖氧化酶反应 → 电极检测H₂O₂电流 → 算法校准得血糖值。与Abbott同类路线，但精度更优

## 技术评估

### 核心技术
- **高精度电化学传感**：MARD 8.0%，行业CGM精度最佳之一，ISO 15167 Zone A+B >99%
- **实时监测生态**：每5分钟自动测量 + Follow App家人远程监控 + Apple Health/Google Fit集成

### 技术壁垒
- FDA/CE双认证，品牌认知度极高
- 与Apple Watch深度集成（直连，无需手机）
- 精度算法积累20+年

## 优势与风险

**优势**
1. MARD 8.0%，CGM精度标杆
2. 每5分钟实时监测，用户体验最佳
3. Apple Watch直连，生态优势明显
4. OTC产品Stelo扩展消费级市场

**风险 / 劣势**
1. 价格高于Abbott 30-40%，性价比不占优
2. 中国NMPA尚未获批，中国市场空白
3. 微创方案，仍有用户依从性门槛
4. 传感器需持续订阅，长期使用成本高

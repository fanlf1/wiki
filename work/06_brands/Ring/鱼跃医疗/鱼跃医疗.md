# 鱼跃医疗

## 一句话定位
> 中国医疗器械龙头，以R3/R3 Pro健康戒指切入消费级市场，R3 Pro是全球首款医疗级透射式SpO2戒指。

## 与我们的关系
- **对标关系**：直接竞品（中国市场）+ 医疗级认证参考
- **关注优先级**：高
- **核心关注点**：R3 Pro的医疗级SpO2认证路径，对我们血糖戒指的NMPA认证有参考价值

### R3 Pro（核心产品）

**核心参数**
| 参数 | 规格 |
|------|------|
| 核心功能 | 透射式SpO2连续监测（**医疗级**） |
| SpO2精度 | 符合医疗器械标准 |
| 其他功能 | 心率、睡眠、活动 |
| 续航 | 7天 |

**技术路线**
- 透射式光电传感（非反射式） → 连续血氧饱和度监测 → 医疗级精度认证。透射式比反射式精度更高，是医疗级认证的技术基础

## 技术评估

### 核心差异化
- **医疗级SpO2认证**：戒指形态中唯一获得医疗器械认证的SpO2产品
- **透射式传感**：精度优于反射式PPG，但功耗更高
- **20年血压测量技术积累**：算法和医疗合规经验丰富

### 与我们（血糖戒指）的关系

| 对比项 | 鱼跃R3 Pro | 我们的血糖戒指 |
|--------|-----------|----------------|
| 认证类型 | SpO2医疗器械 | 血糖（更难） |
| 传感路线 | 透射式光电 | 高光谱CIS |
| 中国渠道 | 药店+电商+医院 | — |
| 品牌背书 | 医疗器械龙头 | — |

## 优势与风险

**优势**
1. 中国医疗器械龙头，医院和药店渠道覆盖最广
2. R3 Pro医疗级SpO2认证，证明戒指形态可以获批医疗器械
3. 深交所上市，资金和供应链成熟
4. 透射式传感精度优势

**风险 / 劣势**
1. 消费级品牌认知度不如Oura
2. 产品功能单一（SpO2为主），生态不如Oura丰富
3. 只在中国市场销售

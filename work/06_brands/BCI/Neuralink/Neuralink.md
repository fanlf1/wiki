# Neuralink

## 一句话定位
> Elon Musk创立的侵入式脑机接口公司，N1植入芯片已完成13例人体植入，目标是从医疗辅助走向大众增强。

## 与我们的关系
- **对标关系**：间接（技术路线和目标市场完全不同）+ 行业标杆
- **关注优先级**：中
- **核心关注点**：Neuralink定义了BCI行业的天花板和公众认知，但不构成直接竞争——我们做非侵入式消费级，他们做侵入式医疗级

### N1芯片（核心产品）

**核心参数**
| 参数 | 规格 |
|------|------|
| 电极数 | 1024个 |
| 植入方式 | 手术机器人植入大脑皮层 |
| 数据传输 | 无线蓝牙 |
| 外形 | 硬币大小，嵌入颅骨 |
| 临床进度 | 13名患者已植入 |

**技术路线**
- 手术机器人将1024电极细丝植入大脑皮层 → 神经元信号采集 → 芯片无线传输 → AI解码。目标是高带宽双向脑机通信

## 技术评估

### 核心技术
- **手术机器人**：自动避开血管精准植入电极，手术速度和精度超越人工
- **超细电极细丝**：比头发丝还细，减少脑组织损伤
- **无线封装**：硬币大小全无线，无外露线缆

### 技术壁垒
- 资金壁垒极高（$6亿+投入）
- FDA临床试验审批（已获突破性设备认定）
- 神经外科手术体系
- Elon Musk个人品牌和资源动员能力

## 优势与风险

**优势**
1. 全球侵入式BCI的绝对领导者
2. $6亿+资金投入，资源远超所有BCI初创
3. 13例成功人体植入，临床数据领先
4. Elon Musk的全球关注度，定义了BCI的公众认知

**风险 / 劣势**
1. 侵入式方案需开颅手术，适用人群极窄
2. 长期生物相容性未知（电极在脑内数年后的状态）
3. 监管路径漫长，离消费级应用可能还需10年+
4. 伦理和公众接受度争议

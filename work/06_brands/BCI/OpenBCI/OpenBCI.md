# OpenBCI

## 一句话定位
> 全球最大的开源BCI平台，以开放硬件+软件生态服务于BCI研究和开发者社区，是BCI领域的"Arduino"。

## 与我们的关系
- **对标关系**：潜在合作（开发工具/参考设计）+ 技术参考
- **关注优先级**：低
- **核心关注点**：其开源EEG硬件可作为我们理解BCI信号采集的参考；如果未来产品涉及脑电功能，OpenBCI生态可能有工具价值

## 技术评估

### 核心技术
- **开源EEG硬件**：完整的原理图、固件、PCB设计全部开源
- **多模态传感**：Galea集成EEG + fNIRS + EMG + PPG，单一设备多维度
- **软件生态**：OpenBCI GUI + LSL集成 + Python/MATLAB SDK

### 技术壁垒
- 开源社区积累（全球最大BCI开发者社区）
- 多模态传感融合设计能力

## 优势与风险

**优势**
1. 开源生态，全球开发者信任度高
2. Galea的多模态（EEG+fNIRS+EMG+PPG）与我们多传感器融合的思路有共鸣
3. 低成本方案降低了BCI研究门槛

**风险 / 劣势**
1. 开源模式盈利能力有限
2. 面向研究者而非消费者，市场规模天花板低
3. 与我们产品方向关联度较低

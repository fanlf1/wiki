**技术路线**
- 碳基柔性干电极 → 耳内接触采集EEG信号 → AI算法解码 → 脑健康状态分析。干电极免导电膏，适合日常佩戴

## 技术评估

### 核心技术
- **碳基柔性干电极**：无需导电膏，长期佩戴舒适，解决了传统EEG湿电极的便捷性痛点
- **耳内多信号融合**：同时采集EEG（脑电）+ EMG（肌电）+ ECG（心电），单一设备多维度健康监测
- **超轻量设计**：≤6g，接近普通入耳式耳机重量

### 技术壁垒
- 碳基柔性干电极材料和工艺
- 耳内微弱EEG信号的提取和降噪算法
- 多信号融合分析模型

## 优势与风险

**优势**
1. 全球最轻消费级BCI（≤6g），佩戴体验领先
2. 碳基柔性干电极，免导电膏，日常可用
3. 多信号融合（EEG+EMG+ECG），单设备多维度
4. CES 2026获奖，品牌曝光强
5. 中国本土团队，供应链和成本优势
6. 已完成众筹并进入国内小规模发售，量产能力初步验证

**风险 / 劣势**
1. 2024年才成立，团队和产品成熟度待验证
2. 耳内EEG信号质量弱于头皮EEG，精度受限
3. 无监管认证，无临床数据
4. A轮1000万人民币融资规模较小，续航能力存疑
5. 小规模发售阶段，量产稳定性和品控待观察

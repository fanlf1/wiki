# 品牌追踪矩阵模板

> **用途**：每个赛道（CGM、BCI等）从本模板派生一份自己的 `brand_matrix.md`。
> 放在赛道目录的根级别，如 `4_brands/CGM/brand_matrix.md`。
>
> 维护者：每次品牌信息有实质更新时同步刷新矩阵。

---

## 填写指南

| 章节 | 作用 | 什么时候改 |
|------|------|-----------|
| 状态说明 | 统一所有矩阵的状态定义 | 不变 |
| 品牌总表 | 一张表看清赛道全貌 | 新增/移除品牌时 |
| 分组视图 | 按需要拆分（如国际/国产、按技术路线等） | 分组方式随赛道特点调整 |
| 关键维度对比 | 可视化核心竞争指标 | 数据更新时 |
| 趋势观察 | 赛道级别的趋势判断 | 季度审视 |
| 信息完整度 | 哪些品牌缺什么 | 每次补充资料后 |
| 下一步 | 补资料的优先级 | 完成一项划掉一项 |

---

## 状态说明

| 状态 | 含义 |
|------|------|
| 深入调研 | 已有详细资料，定期更新 |
| 跟踪中 | 有基本信息，持续关注动态 |
| 暂停关注 | 不再主动跟踪 |

- **按地域**：国际品牌 / 国产品牌

- **按价格带**：高端 / 中端 / 入门
- **按目标用户**：医疗级 / 消费级
- **按产品阶段**：已上市 / 临床 / 原型

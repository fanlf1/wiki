### GBrain - AI Agent 大脑

**定义**：
> "GBrain 赋予 AI 代理一颗大脑。由 Y Combinator 的总裁兼 CEO 打造，用于运行他的实际 AI 代理。"

**核心能力**：

#### 1. 记忆与知识图谱
- **规模**：17,888 页内容
- **实体**：4,383 人
- **公司**：723 家
- **任务**：21 个自动运行的 cron 任务
- **构建时间**：12 天

#### 2. 信息摄取
代理在睡觉时吸收：
- 会议记录
- 电子邮件
- 推文
- 语音通话
- 原创想法

#### 3. 知识连接
每一页的写作提取实体引用，并创建类型化链接：
- `attended`（参加）
- `works_at`（工作于）
- `invested_in`（投资于）
- `founded`（创立）
- `advises`（建议）

**无需调用任何 LLM**，自动完成。

#### 4. 混合搜索
```
Query
  → Intent Classifier (entity? temporal? event? generat?)
  → Multi-query expansion (Claude Haiku)
  → Vector search (HNSW cosine) + Keyword search (tsvector)
  → RRF fusion; score = sum(1/(66 + rank))
  → Cosine re-scoring + Computed truth boost
  → 4-Layer dedup + computed truth guarantee
  → Results
```

**性能提升**：
- 召回率 @5：83% → **95%** (+12个百分点)
- 准确率 @5：39% → **45%** (+6个百分点)
- 前5名中多出 **30个正确答案**
- F1分数：86.6% vs grep的57.8% (+28.8个百分点)

## 🔄 工作原理

### GStack + GBrain 协作

```
┌──────────────────────────────────────┐
│  hosts/gbrain.ts（桥梁）              │
│  告诉 GStack 的编码技能               │
│  在编码前检查大脑                     │
└──────────────────────────────────────┘
         │                    │
    ┌────┴────┐          ┌───┴────┐
    │ GStack  │          │ GBrain │
    │（引擎） │          │（模组） │
    └─────────┘          └─────────┘
```

**协作模式**：
- **GStack**：提供编程技能（开发、评审、质量保证）
- **GBrain**：提供其他一切技能（大脑操作、信号检测、信息摄取）
- **桥梁**：`hosts/gbrain.ts` 告诉 GStack 在编码前检查 GBrain

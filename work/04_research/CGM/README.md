# CGM 评价标准与方法

## 研究问题
> CGM/无创血糖设备的精度如何评价？MARD、ISO 15197等标准是什么含义？我们的无创方案在什么精度水平上可以宣称"临床可用"？

## 关联需求
- REQ-007（血糖趋势追踪，MARD<15%目标）
- REQ-008（异常预警）

## 核心结论
1. MARD < 10%为优秀，< 15%为可用，> 20%不可接受
2. Zone A+B 占比 > 95%是临床安全的基本门槛
3. 睿芯微手持MARD 8.39%已达优秀水平，穿戴版需控制在15%以内

# MARD（平均绝对相对差）精度对比汇总

## 什么是MARD

MARD（Mean Absolute Relative Difference）是衡量血糖监测设备准确度的核心指标，值越低表示精度越高。

### 评价标准

| 等级 | MARD范围 | 说明 |
|------|---------|------|
| 行业领先 | 8.0-8.8% | 接近实验室标准 |
| 主流水平 | 9.0-9.5% | 达到优秀水平 |
| 边缘可接受 | 10-12% | 基本可用 |
| 不可接受 | >12% | 临床价值受限 |

**NMPA指导原则**：合格标准 MARD < 15%，优秀标准 MARD < 10%

## 关键结论

1. **睿芯微MARD 8.39%达到行业领先水平**（手持设备数据，穿戴式一代Demo MARD未公开），与Dexcom G7（8.0%）相当
2. 睿芯微是唯一进入临床验证阶段且MARD<10%的**无创**方案
3. 高血糖区间（≥11.1 mmol/L）MARD低至6.71%，精度优于行业平均
4. 无创+多指标扩展能力是其核心差异化优势

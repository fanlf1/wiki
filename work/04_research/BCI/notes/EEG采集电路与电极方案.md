# Neurable EEG电极方案与采集电路分析

## 一、Neurable合作产品EEG电极配置

### 1.1 MW75 Neuro / MW75 Neuro LT（Master & Dynamic合作）

| 参数         | 规格                                     |
| ---------- | -------------------------------------- |
| **EEG通道数** | 12通道                                   |
| **每耳通道数**  | 6通道                                    |
| **电极类型**   | 干式织物传感器（Dry Fabric Sensor）             |
| **电极材料**   | 编织导电织物（Woven Conductive Fabric）        |
| **电极位置**   | 耳周非标准化位置（Around-Ear, Non-Standardized） |
| **采样率**    | 500 Hz                                 |
| **参考通道**   | Reference + DRL（Driven Right Leg）      |
| **临床标准对比** | 临床EEG帽64+电极，Neurable为12通道              |

**电极位置示意**：

```
          ┌─────────────────────┐
          │      头顶（无电极）     │
          │                     │
          │   颞叶区（接近）       │
          │  ┌───┐       ┌───┐  │
          │  │ L │       │ R │  │
          │  │ 6ch│       │ 6ch│ │
          │  │电极│       │电极│ │
          │  └─┬─┘       └─┬─┘  │
          │    │  耳罩区域   │    │
          │    └─────┬─────┘    │
          │          │          │
          └──────────┴──────────┘

    L6ch ← 左耳罩内6个织物EEG传感器
    R6ch → 右耳罩内6个织物EEG传感器
    Ref  → 参考电极（通常位于耳后或头带）
    DRL  → 驱动右腿电极（共模抑制）
```

**关键特点**：
- 电极嵌入耳机耳垫（Ear Pad）织物中，用户无需额外操作
- 非标准化10-20系统位置，而是围绕耳朵的定制布局
- 干式设计无需导电凝胶，即戴即用
- 12通道虽少于临床64+通道，但Neurable的AI算法弥补了空间分辨率不足

### 1.2 HyperX游戏耳机（CES 2026原型）

| 参数 | 规格 |
|------|------|
| **EEG通道数** | 12通道（与MW75相同平台） |
| **电极类型** | 干式织物传感器 |
| **电极位置** | 耳罩内（Ear Pads） |
| **AI分析** | Neurable AI平台（实时专注力追踪） |
| **新增功能** | 游戏状态优化、Tilt预防、训练习惯分析 |

### 1.3 技术授权方案（通用OEM）

Neurable向第三方授权的EEG技术核心参数：
- **传感器形态**：可集成到耳机、耳塞、头戴等多种外形
- **通道数**：根据产品形态可调（4-16通道）
- **核心价值**：Neurable AI算法平台（基于最大真实世界EEG数据集训练）

### 2.3 ADS1299（TI）典型EEG AFE方案

```
┌─────────────────────────────────────────────────────────┐
│              ADS1299 内部架构（8通道版本）                  │
│                                                          │
│  CH1+ ──→ [PGA] ──→ [ADC] ──→ [数字滤波] ──→ SPI输出    │
│  CH1- ──→ [PGA] ──→ [ADC] ──→ [数字滤波] ──→           │
│  CH2+ ──→ [PGA] ──→ [ADC] ──→ [数字滤波] ──→           │
│  CH2- ──→ [PGA] ──→ [ADC] ──→ [数字滤波] ──→           │
│  ...                                                     │
│  CH8+ ──→ [PGA] ──→ [ADC] ──→ [数字滤波] ──→           │
│  CH8- ──→ [PGA] ──→ [ADC] ──→ [数字滤波] ──→           │
│                                                          │
│  内置功能:                                                │
│  ├── PGA (可编程增益放大器): 1, 2, 4, 6, 8, 12, 24x      │
│  ├── 24位ΔΣ ADC                                          │
│  ├── 输入参考噪声: 1 μVpp (最大增益时)                     │
│  ├── 采样率: 250/500/1000/2000/4000/8000/16000 SPS       │
│  ├── 内置右腿驱动(RLD)放大器                              │
│  ├── 内置测试信号发生器                                   │
│  ├── 掉电/待机模式 (低功耗)                               │
│  └── Daisy-chain 支持 (可扩展至16/24/32通道)               │
│                                                          │
│  电源: AVDD=3.3V, DVDD=1.8V                              │
│  功耗: ~5mW (8通道全开)                                   │
│  封装: TQFP-64                                           │
└─────────────────────────────────────────────────────────┘
```

**注**：Neurable使用12通道方案，可能采用：
- 方案A：2片ADS1299（8ch + 4ch）Daisy-chain
- 方案B：1片ADS1299（8ch）+ 其他4通道AFE
- 方案C：自研ASIC（集成12通道AFE）

## 三、关键器件清单

### 3.1 AFE（模拟前端）器件

| 器件 | 厂商 | 通道 | 分辨率 | 噪声 | 采样率 | 功耗 | 说明 |
|------|------|------|--------|------|--------|------|------|
| **ADS1299** | Texas Instruments | 8ch | 24bit | 1μVpp | 250-16kSPS | ~5mW | EEG专用，内置RLD，最常用 |
| **ADS1294** | Texas Instruments | 4ch | 24bit | 1μVpp | 250-16kSPS | ~3mW | 4通道版本 |
| **ADS1298** | Texas Instruments | 8ch | 24bit | 较高 | 250-16kSPS | ~5mW | ECG/EEG通用 |
| **MAX30003** | Maxim (ADI) | 1ch | 24bit | 低 | 128-2048SPS | ~0.5mW | 单通道低功耗 |
| **AD8232** | Analog Devices | 1ch | — | — | — | ~170μW | ECG AFE，可用于简单EEG |
| **INA2321** | Texas Instruments | 1ch | — | — | — | — | 仪表放大器，用于主动电极方案 |

### 3.2 MCU/SoC（主控）

| 器件 | 厂商 | 架构 | 特点 | 适用场景 |
|------|------|------|------|----------|
| **nRF5340** | Nordic | Cortex-M33双核 | BLE 5.2，低功耗 | 可穿戴EEG |
| **nRF52840** | Nordic | Cortex-M4F | BLE 5.0，USB | MW75类耳机 |
| **STM32L4** | ST | Cortex-M4 | 超低功耗 | 通用可穿戴 |
| **STM32WB** | ST | Cortex-M4+M0 | BLE 5.0 | 无线EEG |
| **ESP32-P4** | Espressif | RISC-V双核 | Wi-Fi+BLE，MIPI CSI | 高端可穿戴 |

### 3.3 电极相关器件

| 组件 | 类型 | 说明 |
|------|------|------|
| **Ag/AgCl电极** | 湿式（临床标准） | 金标准信号质量，需导电凝胶 |
| **导电织物电极** | 干式（Neurable用） | 编织金属纤维，舒适，无需凝胶 |
| **柔性印刷电极** | 干式（新兴） | 丝网印刷，可贴合耳道 |
| **不锈钢/金镀电极** | 干式 | 耐用，但需皮肤处理 |
| **主动电极（Active Electrode）** | 干式+缓冲 | 电极处集成运放缓冲，降低阻抗影响 |

### 3.4 被动器件与电源

| 组件 | 说明 |
|------|------|
| **RC低通滤波** | 抗混叠，通常fc=150-300Hz |
| **DC阻断高通滤波** | 去除电极偏移，通常fc=0.5Hz |
| **ESD保护二极管** | TVS/ESD clamp，保护AFE |
| **LDO稳压器** | 为AFE提供低噪声模拟电源 |
| **锂聚合物电池** | 3.7V，100-500mAh |
| **充电IC** | TP4056等，USB充电 |

## 四、Neurable方案 vs 传统临床方案对比

| 对比维度 | Neurable耳机方案 | 传统临床EEG |
|---------|-----------------|------------|
| **电极类型** | 干式织物 | Ag/AgCl湿式 |
| **通道数** | 12 | 64-256 |
| **电极位置** | 耳周非标准化 | 国际10-20系统 |
| **准备时间** | 即戴即用 | 20-60分钟 |
| **采样率** | 500 Hz | 250-1000 Hz |
| **参考/接地** | DRL + Ref | Linked ears / CZ |
| **信号质量** | 中等（AI增强） | 高 |
| **使用场景** | 日常/工作 | 临床/研究 |
| **成本** | 消费级 | 专业级 |
| **AI信号处理** | ✅ 核心 | 可选 |

## 五、信号处理链路

```
原始EEG信号 (含噪声)
     │
     ▼
┌─────────────┐
│  1. 预处理    │  工频陷波(50/60Hz)、DC去除、基线校正
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  2. 伪迹去除  │  眨眼伪迹(EOG)、肌电伪迹(EMG)、运动伪迹
└──────┬──────┘    → ICA独立成分分析 / 阈值检测
       │
       ▼
┌─────────────┐
│  3. 特征提取  │  频带功率: δ(1-4Hz) θ(4-8Hz) α(8-13Hz) β(13-30Hz) γ(30-100Hz)
└──────┬──────┘    → STFT短时傅里叶变换 / 小波变换
       │
       ▼
┌─────────────┐
│  4. AI推理   │  专注度评分、认知负荷、疲劳检测、脑年龄
└──────┬──────┘    → Neurable ML模型（基于大规模真实EEG训练）
       │
       ▼
┌─────────────┐
│  5. 输出      │  用户App展示：专注力曲线、脑健康报告、工作建议
└─────────────┘
```

# 多标志物光谱检测的可行性（血脂+尿酸）

## 研究问题
> 高光谱/近红外光谱能否同时检测多种生物标志物（血脂、尿酸）？每种标志物需要的光谱波段和检测精度如何？多标志物MoE模型的技术路径是什么？

## 核心结论

1. **血脂NIR检测在血清层面已验证，经皮无创有条件可行**——胆固醇/脂质在1700-1720nm有强特征峰，NIR多组分模型精度已接近临床要求；但需InGaAs传感器（硅基CIS截止~1100nm无法覆盖）
2. **尿酸NIR无创检测目前不可行**——血中浓度太低（μmol级）、NIR特征吸收弱、无大样本临床验证；汗液电化学是更成熟的替代路线
3. **MoE模型路线正确，硬件是关键瓶颈**——睿芯微已规划MoE Transformer架构，学术上已有光谱MoE先例；决定因素是传感器能否从硅CIS升级到InGaAs

## 论文证据

### 论文综述1：NIR血脂检测

- **来源**：多篇（Talanta 2016, Vibrational Spectroscopy 2005, PLOS ONE 2024, Semantic Scholar）
- **核心发现**：
  - 血清层面NIR已可同时定量TC/TG/HDL-C/LDL-C（Talanta 2016）
  - LDL-C的NIR RMS误差0.43 mmol/L，接近临床要求±0.5（Vib Spectro 2005）
  - 血脂关键吸收峰：**~1210nm**（C-H 2nd overtone）、**~1700-1720nm**（C-H combination，最强）、**~2200nm**（C-H combination）
- **对我们的意义**：支持血脂可行，但必须使用InGaAs传感器覆盖1700nm波段
- 完整笔记：[[notes/NIR血脂检测文献综述]]

### 论文综述2：NIR尿酸检测

- **来源**：多篇（J Biosensors 2024, RSC Lab Chip 2024, ScienceDirect 2021/2025）
- **核心发现**：
  - 尿酸在1400-1700nm有NIR吸收特征，但经皮信号极弱
  - 2024年NIR尿酸检测仍停留在概念验证（STM Journals 2024）
  - 汗液电化学（Purdue 2023-2024）和SERS+AI（RSC 2024）路线更成熟
- **对我们的意义**：尿酸NIR无创检测暂不可行，建议远期（3-5年）或切换到汗液路线
- 完整笔记：[[notes/NIR尿酸检测与汗液传感文献综述]]

### 论文综述3：多标志物单传感器+MoE

- **来源**：多篇（Goud 2019, RSC Analyst 2022, arXiv SpecMoE 2025, ResearchGate InGaAs 2024）
- **核心发现**：
  - 血糖+尿酸+胆固醇同时检测已在电化学和光化学中验证
  - MoE模型在光谱分析中已有成功先例（SpecMoE 2025）
  - 单个InGaAs传感器（400-1700nm，>70%效率）可覆盖所有波段
- **对我们的意义**：硬件路线选择（InGaAs vs 硅CIS）是决定性因素
- 完整笔记：[[notes/多标志物单传感器与MoE模型文献综述]]

## 技术可行性评估

| 维度 | 评估 | 说明 |
|------|------|------|
| 理论基础 | **强**（血脂）/ **弱**（尿酸） | 血脂NIR有充分文献支撑；尿酸光谱信号太弱 |
| 信号可获得性 | **有条件** | 血糖（硅CIS够）、血脂（需InGaAs覆盖1700nm）、尿酸（信号不够） |
| 算法成熟度 | **中** | MoE架构有先例，但多标志物光谱解混的工程实现尚无成熟方案 |
| 硬件约束 | **严重** | InGaAs功耗高、成本高；戒指形态的功耗和空间是否允许是核心问题 |
| 监管路径 | **模糊** | 每个新标志物都需要独立的临床验证和认证 |

### 传感器波段需求

```
硅基CIS (400-1100nm)     InGaAs (900-1700nm)
├── 血糖 940-1000nm ✅    ├── 血糖 940-1000nm ✅
├── 血红蛋白 400-600nm ✅  ├── 血脂 1700-1720nm ✅
└── SpO2 等 ✅             ├── 尿酸 1400-1700nm ⚠️（信号弱）
                           └── 甘油三酯 1400-1500nm ✅
```

## 风险与未知

- **风险1：InGaAs功耗**：InGaAs传感器功耗显著高于硅基，戒指15-22mAh电池可能无法支撑连续采集
- **风险2：多标志物光谱解混**：血糖/血脂/尿酸的吸收峰有重叠，MoE能否有效解混尚未验证
- **风险3：个体差异**：不同人皮肤厚度、脂肪分布对1700nm波段影响更大（相比940nm）
- **风险4：认证增量成本**：每个新标志物需要独立的临床验证（N=100+），时间和费用累加
- **未解决**：扩展InGaAs（400-1700nm）在戒指形态中的小型化方案？成本？良率？

## 产品路线建议

```
一代产品（2027-2028）
├── 硅基CIS (400-1100nm)
├── 血糖（核心功能）
└── 血红蛋白/SpO2（附加）

二代产品（2029-2030）
├── InGaAs或扩展CIS (400-1700nm)
├── 血糖（继承）
├── 血脂（TC/LDL/HDL/TG）
└── MoE模型升级

三代产品（2031+）
├── 高灵敏度传感器
├── 血糖+血脂（继承）
├── 尿酸（如果信号问题解决）
└── 更多标志物（肌酐、炎症标志物）
```

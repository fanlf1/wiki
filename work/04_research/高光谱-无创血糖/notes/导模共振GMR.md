# 导模共振（GMR）

## 定义

**GMR** = Guided-Mode Resonance（导模共振）

一种光学共振现象，通过在波导结构中设计特定的光学参数，使得特定波长的光在波导中产生共振，从而实现对光的精确控制和滤波。

## 物理原理

### 基本机制

```
入射光 → 光栅耦合器 → 波导层 → 共振模式
              ↓
        特定波长共振
              ↓
        光谱选择/滤波
```

**核心思想**：
1. 光通过光栅耦合进入波导层
2. 在波导中形成导模
3. 满足共振条件时，特定波长被"捕获"
4. 非共振波长透射或反射
5. 实现光谱分离

### 数学描述

**共振条件**：
```
β·Λ = m·2π
```
其中：
- β = 波导模式的传播常数
- Λ = 光栅周期
- m = 衍射级数（整数）

**共振波长**：
```
λ_res = n_eff·Λ
```
其中：
- λ_res = 共振波长
- n_eff = 有效折射率

## 技术特点

### 优势

1. **高精度光谱选择**
   - 光谱分辨率高
   - 可精确设计中心波长
   - 带宽可调

2. **CMOS工艺兼容** ⭐
   - 可用标准CMOS工艺制造
   - 与集成电路集成
   - 成本可控

3. **物理稳定性**
   - 基于物理结构，非算法补偿
   - 环境漂移小
   - 重复性高

4. **小型化**
   - 片上集成
   - 无需外置滤光片
   - 适合可穿戴设备

### 挑战

1. **设计复杂**
   - 需要精确的光学设计
   - 制造工艺要求高
   - 容差控制严格

2. **角度敏感**
   - 对入射角度敏感
   - 需要角度校准
   - 或设计角度不敏感结构

3. **温度漂移**
   - 热膨胀导致共振波长偏移
   - 需要温度补偿
   - 或温度稳定设计

## 在多光谱成像中的应用

### 睿芯微方案

**片上光学结构（GMR）**：
```
像素级结构：
每个像素单元包含：
- 光栅耦合器
- 波导层
- 共振腔
- 光电探测器

结果：每个像素对应特定光谱波段
```

**与传统方案对比**：

| 维度 | GMR方案 | 滤光片方案 |
|------|---------|-----------|
| **集成度** | 片上集成 | 外置组件 |
| **精度** | 高 | 中 |
| **成本** | 中（批量后低） | 低 |
| **可制造性** | 高（CMOS兼容） | 成熟 |
| **光谱准确性** | 高 | 受环境影响 |
| **算法依赖** | 低-中 | 高 |

## 技术挑战

### 设计挑战

1. **多目标优化**
   - 共振波长 vs 带宽 vs 效率
   - 需要权衡取舍
   - 需要精确仿真

2. **容差管理**
   - 制造误差影响
   - 需要鲁棒设计
   - 或后制程调谐

3. **偏振相关**
   - 某些结构偏振敏感
   - 需要偏振不敏感设计
   - 或偏振分集

### 制造挑战

1. **工艺精度**
   - 光栅周期精度要求高
   - 层厚控制严格
   - 需要先进光刻

2. **对准精度**
   - 层间对准
   - 套刻精度
   - 影响性能

3. **缺陷控制**
   - 缺陷影响共振
   - 需要高良率
   - 增加成本

### GMR vs 其他光谱技术

| 技术 | 原理 | 优势 | 劣势 |

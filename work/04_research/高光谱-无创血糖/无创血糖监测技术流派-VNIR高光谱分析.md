# 无创血糖监测技术流派：VNIR 高光谱分析

睿芯微（Ruixin Micro）采用的 **400-1000nm** 波段方案，在技术分类上属于 **“可见光 - 极近红外光谱分析法 (Visible/Near-Infrared Spectroscopy, VNIR)”** 流派。

## 1. 物理基础与硬件特性

*   **光谱范围**：400nm（可见光紫端）至 1000nm（极近红外端）。
*   **物理原理**：捕捉血液成分的**倍频吸收（Overtones）**和电子跃迁信号。
*   **硬件优势**：该波段与 **标准 CMOS 工艺** 完美兼容，无需使用昂贵的铟镓砷（InGaAs）传感器，可实现“芯片级”集成，成本低且易于大规模量产。
*   **信号源**：对氧合/脱氧血红蛋白、黑色素高度敏感，葡萄糖信号在此区间极弱（被称为“弱信号区”）。

## 2. 核心流派对比：标准型 vs. 高光谱型

在 400-1000nm 流派内，存在两种截然不同的实现路径：

| 维度 | 普通消费级 VNIR (如 dido/HKWN 等) | **高光谱视觉流派 (睿芯微)** |
| :--- | :--- | :--- |
| **通道数量** | 2-4 个离散 LED 波段（窄带） | **100 - 1000+ 个连续光谱通道** |
| **空间维度** | 单点测量，易受佩戴松紧/位置干扰 | **成像测量**，可分辨血管与皮肤层，提取空间特征 |
| **算法复杂度** | 简单线性回归或曲线拟合 | **MoE Transformer** 等深度学习模型 |
| **算力要求** | 低（传统 MCU 即可） | **极高（需 600 GOPS+ 加速器）** |
| **抗干扰力** | 极弱，受肤色和环境光影响大 | **极强**，通过多波段连续光谱抵消环境背景 |

## 3. 全局技术流派横向评估

| 流派名称 | 代表方案/厂商 | 优点 | 缺点 / 挑战 |
| :--- | :--- | :--- | :--- |
| **VNIR 高光谱** | **睿芯微 (ACCU)** | **成本极低、CMOS 兼容**、信息维度极高、支持多指标（血脂/血氧等）。 | **信号极微弱**，对 AI 算法和边缘算力依赖度极高。 |
| **SWIR (短波红外)** | 苹果、三星 (研发中) | 葡萄糖信号强（指纹区）。 | 传感器材质昂贵 (InGaAs)、模组体积大、功耗高。 |
| **拉曼光谱** | Rockley Photonics | 精度上限极高，分子识别性强。 | 需高功率激光器，系统复杂，存在皮肤光安全风险。 |
| **代谢/流变学** | 华为 WATCH 4 | 硬件无需改动（利用现有 PPG），成本最低。 | **非即时监测**，需长佩戴建模，无法给出绝对数值。 |

## 4. 睿芯微流派的战略逻辑

睿芯微方案的核心逻辑是 **“以算力换灵敏度”**：
1.  **回避硬件瓶颈**：不追求昂贵的红外材料，利用成熟的 CMOS 降低门槛。
2.  **升维打击**：利用“高光谱成像”将原本的一维数值测量提升为三维“数据立方体”分析。
3.  **算力兜底**：通过 **600 GOPS** 级别的 Neural-ART 加速器，从海量噪声中精准“提取”出被血红蛋白掩盖的微弱血糖信号。

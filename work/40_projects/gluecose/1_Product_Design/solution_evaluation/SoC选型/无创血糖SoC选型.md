# 无创血糖 SoC 选型

> 睿芯微百万像素 CIS，输出接口 MIPI CSI-2 + DVP，全速带宽 ~480 Mbps。SoC 能否直连，完全取决于 CSI-2 支持。

## CSI-2 支持对比

| SoC               | 制程    | CPU            | NPU 算力    | CSI-2      | DVP  | 直连 CIS | 芯片成本    | 适用形态      |
| ----------------- | ----- | -------------- | --------- | ---------- | ---- | ------ | ------- | --------- |
| **STM32N675**     | ~28nm | M55 @800MHz    | 0.6 TOPS  | ✅ 2-lane   | ✅ 并行 | ✅      | $22-35  | **戒指、手表** |
| **高通 W5+ Gen 1**  | 4nm   | 4×A53 @1.7GHz  | ~1.1 TOPS | ✅ 2×4-lane | ✅    | ✅      | $30-40+ | **高端手表**  |
| **高通 Wear 4100+** | 12nm  | 4×A53 @1.7GHz  | ~1.1 TOPS | ✅          | ✅    | ✅      | $25-35  | 中端手表      |
| **BES2800BP**     | 6nm   | 双M55+双STAR-MC1 | 未公开       | ❌          | ❌    | ❌      | $5-10   | **不适用**   |
| **BES2500P**      | 28nm  | M4F @320MHz    | 无         | ❌          | ❌    | ❌      | $5-10   | **不适用**   |
|                   |       |                |           |            |      |        |         |           |

## 结论

| 产品形态 | 选型 | 一句话理由 |
|---------|------|-----------|
| **戒指** | **STM32N675** | 有 CSI-2，小封装（VFBGA169），0.6 TOPS，MCU 级功耗 |
| **高端手表** | **高通 W5+** | 2×4-lane CSI-2 冗余充足，1.1 TOPS，4nm，Android 生态 |
| 中端手表 | 高通 Wear 4100+ | 成本低于 W5+，CSI-2 满足需求 |

**BES2800BP**：无 CSI-2/DVP，百万像素 CIS 不可直连（SPI/QSPI 带宽不足），排除出无创血糖主控。可作为通信协处理器（BT 5.4 + Wi-Fi 6）配合 STM32N675 使用。

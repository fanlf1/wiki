## 文档说明

### 目的
本文档从**采购甲方**角度，明确无创血糖监测方案的验收标准、测试方法、报告要求，确保技术方案满足产品化需求。

### 适用范围
- 技术方案验收
- 样机测试验收
- 量产前评估
- 供应链审核

### 验收原则
1. **以数据说话**：所有指标需提供客观数据支撑
2. **可重复验证**：测试方法可重复、可追溯
3. **风险可控**：识别技术、合规、供应链风险
4. **符合监管**：满足FDA/NMPA/CE等监管要求

## 第二部分：可操作的验收测试

### 测试一：准确性验证测试

#### 目的
验证MARD、ISO 15167、Clarke误差网格等关键准确性指标

#### 测试方案（甲方执行）

**1. 受试者招募**
- 数量：N=20-30人
- 包含：
  - T1D患者：5人
  - T2D患者：10人
  - 健康人：5-10人
  - 覆盖不同肤色、年龄、BMI

**2. 测试流程**
```
第1天（校准）：
├── 空腹8小时后到达测试中心
├── 采集静脉血（参考值）
├── 佩戴无创CGM设备
└── 同时采集指尖血（校准）

第2-7天（连续监测）：
├── 每日4次指尖血对比（空腹、三餐后2小时）
├── 记录饮食、运动、药物
├── 记录任何设备问题
└── 第7天结束，摘除设备

特别测试：
├── OGTT测试（75g葡萄糖）
├── 运动测试（中等强度30分钟）
└── 低血糖诱导（仅在医疗监督下）
```

**3. 参考标准**
- 静脉血：实验室生化分析仪（金标准）
- 指尖血：已校准的血糖仪（ISO 15167认证）

**4. 数据采集要求**
- 无创CGM：每5分钟记录一次
- 参考血糖：与无创CGM同步记录（±1分钟内）
- 记录所有同步时间戳

**5. 数据分析**
厂家或第三方需提供：
- MARD计算（公式、过程）
- ISO 15167误差网格分析
- Clarke误差网格分析
- 20/20规则符合率
- 按血糖浓度分层分析

**6. 通过标准**
```
✅ 通过：
MARD ≤ 10%
Zone A+B ≥ 95%

⚠️ 条件通过：
MARD ≤ 12% 且 有合理解释
Zone A+B ≥ 90% 且 主要在Zone B

❌ 不通过：
MARD > 12%
Zone A+B < 90%
出现任何Zone D/E
```

### 测试二：重复性与精确性测试

#### 目的
验证设备的重复性和精确性

#### 测试方案（甲方执行）

**1. 重复性测试（同一样本）**
```
方法：
1. 使用血糖模拟液（多个浓度水平）
   ├── 低血糖：3.9 mmol/L
   ├── 正常：5.6、7.0 mmol/L
   └── 高血糖：11.1、16.7 mmol/L

2. 每个浓度测量10次
   ├── 同一设备
   ├── 同一环境
   ├── 短时间内完成
   └── 不重新校准

3. 计算CV%（变异系数）
   CV% = (标准差 / 平均值) × 100%

通过标准：CV% ≤ 5%
```

**2. 重现性测试（不同日）**
```
方法：
1. 连续3天，每天同一时间测量
   ├── 同一样本（血糖模拟液）
   ├── 同一设备
   └── 记录环境条件

2. 计算日间差异
   日间差异 = Max(日均值) - Min(日均值)

通过标准：日间差异 ≤ 7%
```

**3. 设备间一致性**
```
方法：
1. 准备5台设备
2. 同时测量同一样本（3个浓度）
3. 每台测量3次
4. 计算设备间CV%

通过标准：设备间CV% ≤ 8%
```

### 测试三：稳定性与漂移测试

#### 目的
验证设备的长期稳定性和漂移特性

#### 测试方案（甲方执行）

**1. 长期稳定性测试**
```
方法：
1. 选择10名受试者
2. 连续佩戴14天
3. 每日2次指尖血校准/验证
4. 记录每日漂移情况

数据分析：
- 计算每日MARD
- 分析MARD随时间的变化趋势
- 绘制漂移曲线

通过标准：
- 第14天MARD ≤ 第1天MARD + 3%
- 无明显漂移趋势
```

**2. 温度漂移测试**
```
方法：
1. 使用血糖模拟液
2. 在3个温度下测量：
   ├── 15℃（低温）
   ├── 25℃（常温）
   └── 35℃（高温）
3. 每个温度测量5次
4. 计算温度系数

通过标准：
温度漂移 < 3%/10℃
```

**3. 湿度漂移测试**
```
方法：
1. 使用血糖模拟液
2. 在3个湿度下测量：
   ├── 30% RH（干燥）
   ├── 50% RH（适中）
   └── 80% RH（潮湿）
3. 每个湿度测量5次
4. 计算湿度影响

通过标准：
湿度漂移 < 5%
```

### 测试四：实时性与延迟测试

#### 目的
验证设备的实时性和与血浆血糖的时间延迟

#### 测试方案（甲方执行）

**1. 响应时间测试**
```
方法：
1. 受试者佩戴设备
2. 快速饮用含糖饮料（相当于50g葡萄糖）
3. 每1分钟测量一次指尖血
4. 记录设备响应时间

数据分析：
- 找到血糖开始上升的时间点
- 计算设备检测到变化的时间
- 响应时间 = 检测时间 - 实际时间

通过标准：响应时间 < 30秒
```

**2. 时间延迟分析**
```
方法：
1. 进行OGTT测试（0, 15, 30, 60, 90, 120分钟）
2. 同时采集：
   ├── 静脉血（参考值）
   ├── 无创CGM数据
   └── 记录精确时间戳

3. 交叉相关分析
   - 计算不同时间偏移下的相关性
   - 找到最大相关性的时间偏移

通过标准：时间延迟 < 10分钟
```

**3. 低血糖报警延迟**
```
方法：
1. 在医疗监督下进行
2. 诱导轻微低血糖（<3.9 mmol/L）
3. 记录设备报警时间
4. 对比实际低血糖发生时间

通过标准：报警延迟 < 5分钟
```

---

### 测试五：安全性与生物相容性测试

#### 目的
验证设备的安全性和生物相容性

#### 测试方案（甲方或第三方实验室）

**1. 皮肤刺激测试**
```
方法：
1. 20名受试者
2. 佩戴设备24小时
3. 移除后观察皮肤反应
4. 评估标准：红斑、水肿、瘙痒等

评分标准（ISO 10993-10）：
0分：无反应
1分：轻微红斑
2分：明显红斑
3分：严重红斑+水肿
4分：严重反应

通过标准：所有受试者评分 ≤ 1分
```

**2. 温升测试**
```
方法：
1. 受试者佩戴设备
2. 连续测量接触点温度
3. 测量时间：至少2小时
4. 环境温度：25℃

通过标准：最高温度 < 41℃
```

**3. 光辐射安全测试**
```
方法：
1. 测量设备光输出
2. 评估是否符合IEC 62471
3. 测量不同工作距离下的辐照度

通过标准：无风险等级（Risk Group 0）
```

---

### 测试六：可靠性与寿命测试

#### 目的
验证设备的可靠性和使用寿命

#### 测试方案（甲方或第三方实验室）

**1. 传感器寿命测试**
```
方法：
1. 使用模拟使用环境
2. 连续运行设备
3. 每日校准并记录MARD
4. 直到MARD > 15%或设备故障

通过标准：传感器寿命 ≥ 30天
优秀：传感器寿命 ≥ 90天
```

**2. MTBF测试**
```
方法：
1. 抽取20台设备
2. 连续运行1000小时
3. 记录故障次数
4. 计算MTBF

MTBF = 总运行时间 / 故障次数

通过标准：MTBF ≥ 8760小时
```

**3. 环境适应性测试**
```
方法：
1. 跌落测试：
   ├── 1米高度，3次跌落
   └── 功能正常

2. 振动测试：
   ├── 10-500Hz，1小时
   └── 功能正常

3. 温度循环：
   ├── -10℃→40℃→-10℃
   └── 10个循环

通过标准：所有测试后功能正常，MARD不变
```

## 第四部分：验收流程与决策

### 阶段一：文件审核（1-2周）

#### 目的
在样机测试前，审核厂家提交的所有报告

#### 审核清单

**必须通过（否则不进入样机测试）**
- [ ] 临床验证报告完整
- [ ] MARD ≤ 10%
- [ ] ISO 15167 Zone A+B ≥ 95%
- [ ] 电气安全报告齐全
- [ ] 知识产权清晰（无侵权风险）
- [ ] ISO 13485证书有效
- [ ] 批次记录可追溯

**警告项（需合理解释）**
- [ ] MARD在8-10%之间（需说明改进计划）
- [ ] 部分测试报告缺失（需补充时间表）
- [ ] 供应商存在单一来源（需风险说明）

#### 决策
```
✅ 通过：
- 所有必要文件齐全
- 关键指标达标
- 无严重风险

⏳ 条件通过：
- 关键指标达标
- 部分文件缺失但有补充计划
- 非关键指标有偏差但有改进计划

❌ 不通过：
- 关键指标不达标
- 文件严重缺失
- 存在重大风险（IP、安全、合规）
```

### 阶段二：样机测试（4-6周）

#### 测试内容

**1. 实验室测试（1周）**
- [ ] 重复性测试
- [ ] 稳定性测试
- [ ] 温湿度漂移测试
- [ ] 电气安全测试
- [ ] 功能测试

**2. 人体测试（4周）**
- [ ] 招募20-30名受试者
- [ ] 连续监测14天
- [ ] 准确性验证
- [ ] 用户体验评估

**3. 可靠性测试（1周，并行）**
- [ ] 跌落、振动测试
- [ ] 寿命测试（加速）
- [ ] 环境适应性测试

#### 验收标准

**必须通过**
```
实验室测试：
✅ 重复性CV% ≤ 5%
✅ 14天MARD漂移 < 3%
✅ 温度漂移 < 3%/10℃
✅ 功能100%正常

人体测试：
✅ MARD ≤ 10%（90%置信区间上限 ≤ 12%）
✅ ISO 15167 Zone A+B ≥ 95%
✅ 无严重不良事件
✅ 用户满意度 ≥ 4/5

可靠性测试：
✅ 所有测试通过
✅ 无致命故障
```

**条件通过**
```
实验室测试：
⏳ MARD在10-12%之间
⏳ 部分测试有偏差但可解释

人体测试：
⏳ MARD ≤ 12%且有改进计划
⏳ Zone A+B ≥ 90%且主要是Zone B

可靠性测试：
⏳ 非关键测试未通过但可改进
```

**不通过**
```
❌ MARD > 12%
❌ Zone A+B < 90%
❌ 出现Zone D/E
❌ 有严重不良事件
❌ 关键功能失效
❌ 安全测试不通过
```

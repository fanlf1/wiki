# 血糖扩散通路 + 顺序 + 时间关系（极简清晰版）

## 一、**正确流经顺序**

食物糖分→**肠道吸收入血**

1. **门静脉（内脏静脉）** → 2. **肝脏** → 3. **体静脉** → 4. **右心→肺氧合** → 5. **动脉** → 6. **微血管（毛细血管）** → 7. **组织液** → 8. 进入细胞利用

**最简流动顺序：****静脉 → 心脏肺循环 → 动脉 → 微血管 → 组织液**

## 二、时间快慢关系（进食后）

1. **餐后 5–15 分钟**：糖分进入**肠道静脉**，静脉血糖最先升高
2. **15–30 分钟**：经心肺进入**动脉**，动脉血糖上升
3. **30–60 分钟**：动脉血灌流全身**微血管**，微血管血糖达高峰
4. **60–90 分钟**：糖分从微血管渗进**组织液**，供肌肉、脂肪、脏器细胞吸收
5. **2 小时**：组织大量耗糖，全路径血糖整体回落

## 三、浓度梯度（决定扩散方向）

**动脉血糖 ＞ 微血管血糖 ＞ 组织液血糖 ＞ 细胞内糖**顺着浓度差**被动扩散**，不用耗能

## 四、一句话总结

吃完先进**静脉**，心脏换气变**动脉**，动脉分到**微血管**，最后渗进**组织液**被身体用。静脉升最快，动脉次之，组织液最晚达峰值。

# 糖尿病胰岛素异常核心原因（极简透彻版）

## 一、先分清两类最常见

### 1 型糖尿病（年轻人多）

**原因：胰岛彻底受损，几乎不分泌胰岛素**

1. 自身免疫攻击胰腺胰岛 β 细胞
2. 病毒感染、遗传诱发免疫系统错乱
3. 胰岛直接坏死，**身体缺胰岛素**
4. 特点：胰岛素**绝对缺乏**，必须终身打针

### 2 型糖尿病（90% 人都是这个）

**原因：胰岛素抵抗 + 胰岛累垮**这是最核心、最主流病因

1. **胰岛素抵抗（最先出现）**
    
    细胞大门锁死，胰岛素进不去，糖进不了细胞
    
    身体必须**拼命多分泌胰岛素**才能降糖
    
    → 形成**高胰岛素血症**
2. **胰岛超负荷透支**
    
    长年累月疯狂加班分泌
    
    胰岛 β 细胞慢慢疲劳、衰退、减产
3. 最终变成：**胰岛素分泌不足 + 抵抗双重问题**

## 二、导致胰岛素异常的 6 大根本诱因

1. **肥胖（头号元凶）**
    
    内脏脂肪直接堵住细胞受体，产生强抵抗
2. **长期高糖高油饮食**
    
    持续刺激胰岛超负荷工作，提前早衰
3. **久坐不运动**
    
    肌肉不消耗糖，受体敏感度暴跌
4. **熬夜、压力大**
    
    升糖激素飙升，抵消胰岛素作用
5. **遗传体质**
    
    天生胰岛储备弱、易发生抵抗
6. **年龄增长**
    
    胰岛功能自然逐年下降

## 三、胰岛素异常三种状态

1. **分泌太多**：早期 2 型，抵抗重，胰岛素偏高
2. **分泌延迟**：胰岛反应慢，餐后血糖冲高、下一餐低血糖
3. **分泌太少**：中晚期 2 型 + 1 型，胰岛功能衰竭

## 四、一句话总结

**早期是身体不听胰岛素使唤（抵抗），后期是胰腺造不出胰岛素（衰竭），两者叠加就是糖尿病。**

## 五、简单调理逻辑

- 减体重、控饮食 → 减轻**胰岛素抵抗**
- 规律运动 → 打开细胞糖门
- 早睡减压 → 保护胰岛不再继续衰退

# 肥胖核心成因（权威溯源 + 分类清晰）

## 一、核心总原则

**能量摄入＞能量消耗**，多余热量以脂肪形式储存，是肥胖根本机制，出自《营养学》《肥胖病学》通用定论。

## 二、大类成因 + 权威依据

### 1. 饮食因素（最主要后天原因）

1. **热量超标**
    
    高油、高糖、精加工食品、外卖、甜食、含糖饮料，热量密度极高，远超日常所需
    
    出处：《中国居民膳食指南 2022》，明确不合理膳食是超重肥胖首要饮食诱因
2. **饮食结构失衡**
    
    主食精米白面过多，蔬果粗粮不足，蛋白质偏少
3. **进食习惯差**
    
    暴饮暴食、夜宵、三餐不规律、吃饭过快、睡前进食

### 2. 运动与生活方式

1. **久坐少动**
    
    办公、刷手机、出行代步，日常活动量断崖式下降，基础消耗大幅降低
    
    出处：WHO 慢病报告，久坐行为独立提升肥胖风险 2 倍以上
2. **缺乏规律运动**
    
    无力量 + 有氧训练，肌肉量偏低，基础代谢偏弱

### 3. 遗传与先天体质

1. **家族遗传**
    
    父母肥胖，子女肥胖概率提升 50%~70%，影响脂肪分布、基础代谢、食欲激素
2. **先天代谢差异**
    
    部分人群天生脂肪囤积倾向更强，脂肪分解效率更低
    
    出处：《Nature Reviews Endocrinology》肥胖遗传机制综述

### 4. 内分泌与激素紊乱（直接致胖）

1. **胰岛素偏高 / 胰岛素抵抗**
    
    血糖波动大，身体极易囤积脂肪，尤其腹部内脏脂肪（和你前面聊的糖尿病直接联动）
2. **皮质醇偏高**
    
    长期熬夜、压力大，压力激素升高，优先堆积腰腹脂肪
3. **甲状腺功能减退**
    
    代谢变慢，怕冷乏力、体重莫名上涨
4. **多囊卵巢综合征**
    
    女性高发，伴随发胖、血糖异常

### 5. 睡眠与情绪因素

1. **长期熬夜、睡眠不足**
    
    扰乱瘦素、饥饿素分泌：饱腹激素减少，饥饿激素升高，食欲暴涨偏爱高热量食物
2. **情绪性进食**
    
    焦虑、压抑、无聊靠吃东西缓解，无意识超额进食

### 6. 药物与疾病继发肥胖

- 激素类药物、部分抗抑郁药、降糖药等副作用致体重上升
- 脑部病变、代谢类疾病继发脂肪堆积

### 7. 年龄因素

成年后每增长 10 岁，基础代谢自然下降 3%~5%，同等饮食下更易发胖

1. **底层根本**：遗传 + 基础代谢
2. **主导推手**：吃得多、动得少

# JBD（显耀显示）

## 一句话定位
> 全球MicroLED微显示领域头部企业，产品已应用于近50款智能眼镜，B2轮融资超十亿元创全球纪录——是AR眼镜"中国芯"的核心供应商。

## 公司概况

| 项目 | 内容 |
|------|------|
| 全称 | 上海显耀显示科技股份有限公司 |
| 英文 | Jade Bird Display (JBD) |
| 成立时间 | 2015年 |
| 总部 | 上海 |
| 官网 | https://www.jb-display.com.cn/ |
| 核心业务 | MicroLED芯片和面板的研发及生产 |
| 产品线 | 微显示屏、光引擎、光模组 |

## 融资历程

| 轮次 | 时间 | 金额 | 投资方 |
|------|------|------|--------|
| A3轮 | 2022-08 | 数亿元 | 光速中国、斯道资本联合领投 |
| Pre-B轮 | — | 数亿元 | **蚂蚁集团领投**，吉利资本跟投 |
| **B2轮** | **2025-10** | **超十亿元** | 刷新全球MicroLED微显示领域单轮融资纪录 |

## 核心产品

### MicroLED微显示屏

| 参数 | 规格 |
|------|------|
| 芯片尺寸 | 0.13英寸（0.13" VGA） |
| 分辨率 | 640×480 |
| 显示模式 | 单色绿光（Monochrome Green） |
| 最大亮度 | ~1,200 nits |
| 刷新率 | 60 Hz |
| 特点 | 高光效、体积小（光引擎0.2cc/0.5g） |

### 光引擎产品

- 微显示屏 + 微型光学镜片组成完整光引擎
- 适用于AR眼镜铰链处安装
- 已支持近50款智能眼镜上市（截至2025年底）

## 技术路线

- 当前：单色绿光MicroLED（量产成熟）
- 下一代：计划量产"蜂鸟Ⅱ"彩色光引擎和"凤凰系列"单片全彩微显示屏
- 与Dispelix有R&D合作伙伴关系（光引擎+光波导联合方案）

## 光波导行业生态（JBD的下游合作伙伴）

| 供应商 | 技术路线 | 量产能力 | 2025动态 |
|--------|----------|----------|----------|
| **歌尔光学** | SRG纳米压印 | 最高 | 舜宇19亿入股，行业整合 |
| **舜宇奥来** | 超表面/衍射 | 晶圆级 | 并入歌尔光学 |
| **鲲游光电** | VHG体全息+SRG+碳化硅 | IDM闭环 | 估值21亿，6大产品系列 |
| **Dispelix** | SRG刻蚀工艺 | 瑞声代工 | Selva 2.0单层全彩 |

> 2025年主流方案已明确为基于纳米压印的SRG衍射光波导。

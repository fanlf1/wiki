## 团队背景

- **创始人孟祥峰**：清华校友，致力于用光栅技术搭建"元宇宙之窗"
- 七年技术沉淀（2019年至今）

## 技术路线：SRG + 纳米压印（NIL）

### 核心技术：表面浮雕光栅（SRG）衍射光波导
- 采用**纳米压印（Nano Imprint Lithography, NIL）**作为核心量产工艺
- 自主掌握三大核心技术：
  1. **光栅设计**：渐变光栅专利技术（最早研发并量产）
  2. **光栅母版加工**：高精度母版制作能力
  3. **纳米压印生产**：全链条自主可控（设备、工艺、材料）

### 渐变光栅技术 — 核心差异化
- 至格科技**最早研发并量产渐变光栅**
- 相当于将传统的3-4个光栅分区扩展为**无数个光栅分区**
- 通过逐点调整每一个点的光效，弥补亮度和色彩的不均匀性
- 有效控制彩虹纹问题

### 工艺能力
- 可实现直齿形、斜齿形、三角形等多种光栅结构的纳米压印
- 压印精度达纳米级
- 材料以树脂/玻璃为主

### 技术判断
- 至格认为：**30度以内FOV的全彩衍射光波导，仍将以纳米压印（NIL）为主**
- 纳米压印在低成本、大批量消费级AR产品中具有不可替代的优势

## 与歌尔光学的差异化

| 维度 | 至格科技 | 歌尔光学 |
|------|---------|---------|
| 出身 | 清华大学孵化 | 歌尔股份控股（2025年与舜宇奥来合并） |
| 核心工艺 | **纳米压印（NIL）**，坚持极致优化 | 从纳米压印升级到**干法刻蚀** |
| 技术特色 | **渐变光栅**专利壁垒，逐点光效调节 | 刻蚀工艺突破高折射率材料瓶颈 |
| 全彩进展 | 以单色/双色为主，全彩在研 | 已推出全彩Star G-E1模组 |
| 量产成熟度 | 高，适合消费级低成本量产 | 刻蚀工艺正在规模推广 |
| 成本 | 纳米压印成本较低 | 刻蚀工艺成本较高但性能更优 |
| 客户绑定 | OPPO、小米战略投资 | 依托歌尔+舜宇资源 |
| 材料路线 | 树脂/玻璃 | 支持碳化硅等高折射率材料 |
| 独特技术 | 渐变光栅 | 碳化硅50°大视场角 |

**核心差异**：至格科技走"成本驱动、量产先行"路线，歌尔光学走"性能驱动、技术升级"路线。

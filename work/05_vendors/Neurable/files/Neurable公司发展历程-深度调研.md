## 二、发展时间线

### 第一阶段：学术起源（2015年）

```
2015
 │
 │  Ramses Alcaide在密歇根大学直接脑接口实验室
 │  完成BCI核心技术研究
 │
 ▼
 正式创立Neurable
 定位：AI软件公司，利用脑传感技术将脑信号转化为可操作洞察
```

**关键技术积累**：
- EEG信号处理算法
- 实时脑信号解码
- AI驱动的脑状态分类

### 第二阶段：技术验证与早期产品（2015-2020）

```
2015 ──── 2016 ──── 2017 ──── 2018 ──── 2019 ──── 2020
  │         │         │         │         │         │
  │         │         │         │         │         │
  创立      技术开发    原型开发    Enten概念    技术迭代    技术验证
                      VR BCI     头戴耳机     算法优化    白皮书发布
```

**关键里程碑**：
- **Enten耳机**：早期概念产品，BCI头戴式耳机原型
- **白皮书**：发布《Neurable Enten Earphone EEG Focus Estimation》，展示干电极系统Quick20
- **核心技术**：开发AI算法平台，基于真实世界EEG数据集训练

**技术方向**：
- 从VR/游戏场景的BCI交互（意念控制）
- 转向消费级专注力追踪和认知健康

### 第三阶段：产品化与商业化（2020-2024）

```
2020 ──── 2021 ──── 2022 ──── 2023 ──── 2024
  │         │         │         │         │
  │         │         │         │         │
  技术转型   算法平台   硬件集成   合作谈判   MW75 Neuro发布
  消费级     成型       耳机形态   寻找合作   (2024.9.24)
  专注力                          音频品牌
```

**重大转折**：与Master & Dynamic达成合作

- 战略决策：不自行生产硬件，而是与高端音频品牌合作
- 将BCI技术集成到高端耳机的耳垫中
- 实现了"你甚至不需要做任何额外操作"的无感体验

**2024年9月24日**：MW75 Neuro正式发布

- 全球首款消费级BCI耳机
- 12通道EEG干织物传感器
- 实时专注力追踪
- 被评为2024年神经科技产品亮点

### 第四阶段：规模化与平台化（2025-2026）

```
2025初 ── 2025中 ── 2025.12 ── 2026.1 ── 2026.4 ── 2026.5
  │         │         │          │         │         │
  │         │         │          │         │         │
  MW75 LT   App重构   $35M A轮   CES 2026  技术授权   继续扩张
  发布      升级      融资       HyperX    平台发布
```

**2025年关键事件**：
- **MW75 Neuro LT发布**：更轻更便宜，使用相同算法
- **Neurable App重构**：全新设计，专注用户体验
- **2025年12月19日**：**完成$3500万A轮融资**

**2026年关键事件**：
- **CES 2026**：HyperX合作脑追踪游戏耳机获最高荣誉
- **2026年4月28日**：TechCrunch报道正式推出技术授权平台
- **Neurable Veo**：扩展BCI平台面向更广泛的消费级可穿戴设备

## 六、技术突破

### 突破时间线

| 时间            | 突破            | 意义                |
| ------------- | ------------- | ----------------- |
| **2015-2017** | 干式织物EEG传感器    | 无需凝胶，即戴即用         |
| **2017-2019** | AI脑信号解码算法     | 从噪声中提取有意义的认知指标    |
| **2019-2021** | 大规模真实世界EEG数据集 | 业内最大，训练AI模型       |
| **2021-2023** | 耳机形态BCI集成     | 将EEG传感器嵌入耳机耳垫织物   |
| **2024**      | 消费级专注力追踪算法    | 实时、准确、无需主动配合      |
| **2025**      | 跨品类BCI平台化     | 可授权到耳机、耳塞、腕戴等多种形态 |

### 核心技术壁垒

1. **干式织物EEG传感器**：编织导电纤维，无凝胶，舒适佩戴
2. **Neurable AI算法**：基于最大真实世界EEG数据集训练
3. **非标准化电极位置补偿**：12通道达到接近临床级效果
4. **无缝集成方案**：传感器嵌入普通耳机外观，用户无感

## 七、公司策略方向演进

### 三次战略转型

```
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│  转型1 (2015→2020)                                           │
│  ┌──────────────┐     ┌──────────────────┐                  │
│  │ 研究型BCI    │ ──→ │ 消费级专注力追踪  │                  │
│  │ VR意念交互   │     │ 日常可用          │                  │
│  └──────────────┘     └──────────────────┘                  │
│       ↓ 目标：学术研究        ↓ 目标：个人消费者              │
│                                                              │
│  转型2 (2020→2024)                                           │
│  ┌──────────────────┐     ┌──────────────────┐              │
│  │ AI软件平台      │ ──→ │ 硬件+软件一体化  │              │
│  │ 纯技术授权     │     │ MW75 Neuro产品   │              │
│  └──────────────────┘     └──────────────────┘              │
│       ↓ 目标：技术供应商      ↓ 目标：消费品牌               │
│                                                              │
│  转型3 (2025→2026)                                           │
│  ┌──────────────────┐     ┌──────────────────┐              │
│  │ 单一产品公司    │ ──→ │ 技术平台公司     │              │
│  │ MW75 Neuro系列  │     │ 授权+自研双轨   │              │
│  └──────────────────┘     └──────────────────┘              │
│       ↓ 目标：耳机消费者      ↓ 目标：全品类可穿戴            │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### 当前策略（2026）

**"从可穿戴到可思考"（Wearables to Thinkables）**

1. **自有产品线**：MW75 Neuro / MW75 Neuro LT持续迭代
2. **品牌合作**：HyperX（游戏）、Master & Dynamic（高端音频）
3. **技术授权**：向OEM厂商开放Neurable AI平台
4. **纵向脑追踪**：长期脑健康数据收集和分析

### 竞争护城河

| 护城河 | 说明 |
|--------|------|
| **AI数据优势** | 最大真实世界EEG数据集，难以复制 |
| **形态创新** | 干织物传感器+耳机集成，用户体验碾压传统BCI |
| **先发优势** | 消费级BCI耳机第一个上市产品 |
| **合作生态** | HyperX(HP)+Master & Dynamic，多品类覆盖 |
| **平台效应** | 从产品到平台，规模效应加速 |

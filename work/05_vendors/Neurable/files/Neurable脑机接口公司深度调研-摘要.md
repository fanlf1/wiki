## 核心内容

### 公司概况

| 项目 | 内容 |
|------|------|
| 公司名称 | Neurable Inc. |
| 成立时间 | 2015年 |
| 创始人 | Dr. Ramses Alcaide（神经科学家） |
| 总部 | 美国波士顿 |
| 技术路线 | 非侵入式脑机接口（EEG + AI） |
| 目标市场 | 消费级市场 |

### 核心技术

**非侵入式BCI技术**：
- 使用脑电图（EEG）传感器读取大脑神经元电活动
- 无需手术植入
- AI算法处理和解读脑信号模式

**技术参数**：
- 12通道EEG传感器
- 软织物传感器设计
- 干传感器技术

### 产品线

| 产品 | 特点 |
|------|------|
| MW75 Neuro | 全球首款消费级BCI耳机，实时脑电波追踪，专注度监测 |
| MW75 Neuro LT | 轻量版，更实惠，同样具备脑传感技术 |

### 商业模式

**B2B2C授权模式**：
- 向OEM厂商授权脑传感技术
- 与Master & Dynamic合作推出MW75 Neuro系列

### 应用场景

1. **专注度管理**：数值化专注度评分，工作效率优化
2. **ADHD支持**：专注力训练辅助，脑健康追踪
3. **生产力提升**："Work Smarter, Not Longer"

### 与Neuralink对比

| 对比维度 | Neurable | Neuralink |
|---------|----------|-----------|
| 技术路线 | 非侵入式（EEG + AI） | 侵入式（植入大脑皮层） |
| 目标市场 | 消费级市场 | 医疗/严肃医疗市场 |
| 产品形态 | 耳机等可穿戴设备 | N1植入芯片 |
| 商业化进度 | 已有消费产品 | 临床试验阶段 |
| 2026计划 | 市场扩张 | 大规模量产 |

### 优势与挑战

**优势**：
- 技术安全性高（非侵入式）
- 商业化进度领先
- 应用场景广泛
- 技术可扩展性强

**挑战**：
- 信号精度限制
- 市场竞争加剧
- 用户接受度
- 技术瓶颈

### 2026年行业趋势

- 中国批准全球首个侵入式BCI医疗器械商业上市
- Neuralink计划从临床试验转向大规模生产
- 中国宣布116亿元脑科学基金
- 2026年被描述为脑机接口的关键之年

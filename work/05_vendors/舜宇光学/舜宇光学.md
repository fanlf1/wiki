## 2025年歌尔光学交易

| 项目 | 详情 |
|------|------|
| 公告 | 2025-08-22 |
| 签署 | 2025-10-09 |
| 交割 | 2025-12-08 |
| 作价 | **19.03 亿元** |
| 方式 | 换股（上海奥来 100% 股权 → 歌尔光学新增股份） |
| 交易后股权 | 舜宇系 ~1/3，歌尔系 ~2/3 |
| 增资 | 双方各向歌尔光学增资 2 亿元 |
| 结果 | 上海奥来 → 歌尔光学全资子公司；歌尔光学拟更名"歌尔奥来光学" |

## 战略意义

- **"新台积电"模式**：专业光学器件代工平台
- **技术互补**：舜宇奥来微纳光学 + 歌尔 XR 光波导量产 + 歌尔股份整机客户（Meta、小米）
- **加速降本量产**：两大果链巨头整合"技术+产能+客户"，避免重复投资
- AI 眼镜 2024 年全球销量超 200 万台，预期 2026 年达千万量级

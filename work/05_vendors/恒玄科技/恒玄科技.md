# 恒玄科技

## 一句话定位
> 专注可穿戴设备和IoT芯片设计的中国Fabless公司，BES2500/BES2800系列SoC是智能手环和TWS耳机的主流方案，以超低功耗和成本优势著称。

## 与我们的关系
- **合作状态**：SoC候选供应商
- **关注优先级**：高
- **核心关注点**：BES2800BP是从MCU级向应用处理器级跨越的里程碑产品，可能适合做健康手表的SoC平台

## 技术评估

### 优势
- 成本最低（5-10美元），BOM仅15-25美元
- 超低功耗，手环14天续航
- BES2800BP性能飞跃：NPU + Wi-Fi 6 + MIPI DSI
- 中国本土公司，支持快速

### 劣势
- BES2500性能有限（MCU级），不适合复杂AI模型
- 生态薄弱，SDK和开发者资源少
- 不支持Wear OS等成熟系统

### 血糖方案适配

| 方案 | 适配性 | 说明 |
|------|--------|------|
| BES2500 + PPG血糖 | ❌ | 性能不足，无法运行AI模型 |
| BES2500 + 外接AI芯片 | ⚠️ | 可行但增加复杂度和成本 |
| BES2800BP + 高光谱CIS | ✅ | NPU可加速推理，需验证 |

# 高光谱成像技术 - 快速参考

## 核心概念（1分钟理解）

### 什么是高光谱成像？

```
传统照片：每个像素 = (红, 绿, 蓝) → 3个数值
高光谱：每个像素 = (λ₁, λ₂, ..., λ₂₀₀) → 200+个数值

→ 每个像素都有一条完整的光谱曲线！
→ 可以分析每个像素点的物质成分
```

### 为什么能检测血糖？

```
葡萄糖分子 → 吸收特定波长的近红外光
→ 吸收强度 ∝ 葡萄糖浓度
→ 通过分析吸收光谱 → 计算血糖值
```


## AI的作用

### 为什么需要AI？

```
高光谱数据特点：
• 640×480像素 × 200波段 = 61,440,000个数值
• 噪声干扰多
• 个体差异大
• 光谱重叠复杂

AI的优势：
✓ 自动特征提取（从61M数据中找关键信息）
✓ 非线性建模（学习复杂映射关系）
✓ 多模态融合（结合图像+光谱+时间序列）
✓ 个性化适应（学习个体特征）
```

### MoE Transformer架构

```
预训练阶段：
大规模未标注血液光谱
        ↓
  Transformer 基础模型
（学习通用血液光谱知识）

微调阶段：
┌────────┬────────┬────────┐
│ 血糖   │ 血脂   │ 血肌酐  │
│ Expert │ Expert │ Expert │
└────────┴────────┴────────┘
    ↓        ↓        ↓
多生物标志物检测
```

**优势：**
- 预训练减少标注数据需求
- Expert分工提高准确性
- 知识迁移加速新指标开发

## 单位换算（重要）

```python
# 中国常用：mmol/L
# 国际常用：mg/dL

mmol/L = mg/dL ÷ 18
mg/dL = mmol/L × 18

# 常见对照
70 mg/dL  ≈ 3.9 mmol/L  (低血糖下限)
100 mg/dL ≈ 5.6 mmol/L  (正常空腹上限)
140 mg/dL ≈ 7.8 mmol/L  (餐后血糖上限)
180 mg/dL ≈ 10.0 mmol/L (高血糖警戒)

# 睿芯微数据
平均偏差: 0.3 mmol/L = 5.4 mg/dL
```

# 无创血糖监测方案对比：睿芯微 vs 多感科技

对比日期：2026-04-13
资料来源：两家厂商技术文档

## 一、核心技术路线对比

| 对比项 | 睿芯微 | 多感科技 |
|--------|--------|----------|
| **技术原理** | 高光谱 CMOS 图像传感器（CIS）+ 血管成像 | 近红外光谱透射/散射分析 |
| **检测方式** | 血管实时成像（类似"智能相机"） | 检测葡萄糖分子O-H和C-H基团的光谱吸收 |
| **光谱范围** | 可见光 + 近红外（高光谱成像） | 近红外 780～2500 nm（重点900-1750nm） |
| **传感器类型** | 高光谱CIS（芯片级光谱成像同步采集） | 硅基分光芯片 + InGaAs线阵传感器 |
| **通道数** | 图像形式（千-万级变量） | 112通道近红外多光谱 |
| **检测位置** | 静脉端（血管成像） | 皮肤透射/散射（组织间液） |

### 技术原理详解

**睿芯微 - 高光谱成像方案：**
```
光学感知系统 → 血管实时成像 → 高光谱CIS采集 → 图像+光谱数据 → AI分析
                                    ↓
                          类似"智能相机"直接观测血管内血糖
```
- 优势：直接检测血管内血糖，理论准确性更高
- 挑战：对光学系统精度要求高，需要穿透皮肤组织

**多感科技 - 近红外光谱方案：**
```
近红外光源 → 皮肤组织 → 硅基分光芯片 → InGaAs传感器 → 112通道光谱数据
                                    ↓
                    检测葡萄糖分子O-H/C-H基团的光谱吸收特征（780-2500nm）
```
- 优势：技术原理清晰，硬件工艺成熟
- 挑战：需要通过组织间液推断血糖，信号衰减大

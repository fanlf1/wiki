# Obsidian使用指南 - LLM Wiki可视化

## 📖 什么是Obsidian？

**Obsidian**是一个基于本地Markdown文件的知识库工具，非常适合查看LLM Wiki：

- ✅ **双向链接**：自动识别`[[链接]]`格式
- ✅ **图谱视图**：可视化知识网络
- ✅ **实时预览**：LLM修改文件后立即看到
- ✅ **反向链接**：查看哪些页面链接到当前页
- ✅ **全文搜索**：快速找到内容
- ✅ **本地存储**：数据完全在你的控制下

---

## 🎯 核心功能

### 1. 文件导航（左侧栏）

**文件列表**：
- 查看所有markdown文件
- 点击文件名打开

**搜索**（Ctrl+Shift+F）：
- 全文搜索所有文件
- 支持正则表达式

### 2. 链接跳转

**示例**：
```markdown
# 当前页面：index.md

- [[MARD]] — 点击跳转到MARD概念页
- [[睿芯微]] — 点击跳转到睿芯微实体页
```

**操作**：
- 点击`[[链接]]` → 跳转到目标页面
- Ctrl+点击 → 在新面板打开
- 右键 → 选择"在右侧打开"

### 3. 反向链接（右侧栏）

**功能**：查看哪些页面链接到当前页面

**示例**：
- 打开`MARD.md`
- 右侧栏显示：
  - `CGM方案总结-摘要.md` 提到MARD
  - `睿芯微.md` 引用MARD数据
  - `CGM评价标准-摘要.md` 解释MARD

### 4. 图谱视图（Graph View）⭐

**打开方式**：
- 点击左侧栏**"图谱视图"图标**（🕸️）

**功能**：
- 可视化所有页面和链接关系
- 节点 = 页面
- 连线 = 链接关系
- 点击节点 → 打开页面

**操作**：
- 滚轮缩放
- 拖拽移动
- 点击节点查看详情
- 悬停显示预览

### 5. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+O` | 快速打开文件 |
| `Ctrl+Shift+F` | 全局搜索 |
| `Ctrl+E` | 切换预览/编辑模式 |
| `Ctrl+K` | 插入链接 |
| `Ctrl+G` | 打开图谱视图 |
| `Ctrl+\` | 切换侧边栏 |
| `Alt+←` | 返回上一页 |

## 🔄 与LLM Wiki配合使用

### 推荐工作流

**左侧：Claude Code对话** | **右侧：Obsidian实时查看**

```
Claude Code                    Obsidian
     ↓                              ↓
告诉LLM："摄取新源"  ←→  实时看到文件更新
     ↓                              ↓
讨论关键要点        ←→  浏览现有相关页面
     ↓                              ↓
LLM创建新页面       ←→  点击链接探索
     ↓                              ↓
更新索引           ←→  图谱视图看变化
```

### 具体操作

**1. 摄取时**
```bash
# 在Claude Code中
"摄取 raw/notes/solution/新方案.md"
```
→ Obsidian中自动看到新文件出现

**2. 查询时**
```
# 在Claude Code中提问
"睿芯微和Dexcom G6的MARD对比？"
```
→ LLM会读取相关页面 → 你可以在Obsidian中打开这些页面对照查看

**3. 探索时**
```
# 在Obsidian图谱视图中
点击"睿芯微"节点
→ 查看所有相关链接
→ 点击"MARD"节点
→ 继续探索...
```

## 🎨 个性化设置

### 推荐插件

**核心插件**（内置）：
1. ✅ **反向链接** — 必开
2. ✅ **图谱视图** — 必开
3. ✅ **大纲** — 建议开启
4. ✅ **搜索** — 必开

**第三方插件**（可选）：
1. **Dataview** — 查询语言，可统计页面数据
2. **Advanced Tables** — 更好的表格编辑
3. **Calendar** - 日历视图
4. **Kanban** - 看板视图

**安装插件**：
1. 设置 → 第三方插件 → 浏览
2. 搜索插件名称
3. 安装并启用

### 主题推荐

**默认主题**就很好用，或者尝试：
- **Minimal** - 简洁现代
- **Things** - 清新风格
- **Primary** - 主色调主题

**切换主题**：
设置 → 外观 → 主题

## 💡 最佳实践

### 1. 实时查看LLM更新

LLM编辑文件时：
- Obsidian会自动检测文件变化
- 点击**"重新加载文件"**提示
- 或按`Ctrl+R`手动刷新

### 2. 双屏布局

**水平分割**（Ctrl+\）：
```
┌──────────────────┬──────────────────┐
│  index.md        │  MARD.md         │
│  (索引)          │  (详情)          │
└──────────────────┴──────────────────┘
```

**垂直分割**：
```
┌──────────────────────────────────────┐
│  index.md          (索引)            │
├──────────────────────────────────────┤
│  MARD.md           (详情)            │
└──────────────────────────────────────┘
```

### 3. 快速导航

**场景**：从MARD跳到睿芯微，再跳到无创血糖监测

1. 在`MARD.md`中点击`[[睿芯微]]`
2. 在`睿芯微.md`中点击`[[无创血糖监测]]`
3. 点击工具栏"返回"按钮回到MARD

### 4. 全局搜索

**场景**：查找所有提到"8.39%"的页面

1. Ctrl+Shift+F打开搜索
2. 输入：`8.39%`
3. 查看所有匹配结果

## 🆘 常见问题

### Q1：文件未显示

**解决**：
- 检查左侧文件列表
- 点击"刷新"按钮
- 或按`Ctrl+R`

### Q2：链接点击无效

**解决**：
- 确保目标文件存在
- 检查文件名是否正确（区分大小写）
- 链接格式：`[[文件名]]`

### Q3：图谱视图没有节点

**解决**：
- 等待Obsidian索引文件
- 点击图谱视图设置
- 勾选"显示孤立节点"

### Q4：中文文件名乱码

**解决**：
- 确保文件编码为UTF-8
- 设置 → 编辑器 → 默认编码 → UTF-8

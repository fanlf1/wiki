# 法布里-珀罗干涉（Fabry-Perot Interference）

## 定义

**法布里-珀罗干涉**是一种基于多光束干涉原理的光学技术，通过精确设计的多层结构实现光谱分离，选择性通过特定波长同时阻挡其他波长。

## 基本原理

```
物理机制：
├── 两块平行半透明镜面
├── 光线在镜面间多次反射
├── 多光束干涉形成谐振
└── 只有特定波长的光能透过

核心参数：
├── 镜面反射率：决定干涉峰的锐度
├── 镜面间距：决定中心波长
├── 介质折射率：影响光程差
└── 入射角度：影响波长选择
```

## 在多光谱成像中的应用

### 干涉滤光片

```
实现方式：
├── 精确设计的多层结构
├── 控制叠层厚度决定通过波长
├── 选择性透过特定波长
└── 阻挡其他波长

技术特点：
├── 窄带光谱响应（通常<10 nm）
├── 高光谱分辨率
├── 优异的稳定性
├── 出色的光谱重建能力
└── 可集成到芯片上

vs 有机滤光片：
├── 有机：光谱响应宽（>100 nm），稳定性差
└── 干涉：窄带响应，高稳定性
```

### Spectricity的法鉑腔精密滤波技术

```
技术特点：
├── 基于法布里-珀罗干涉原理
├── 马赛克形式分布
├── 通过控制CMOS传感器上滤光层厚度决定中心波长
├── 每个像素都具备窄带光谱响应
├── 可扩展并适配任何图像传感器
└── 将多光谱成像带入超小型化领域

专利保护：
└── 法鉑腔精密滤波技术（已获专利）

产品实现：
├── Spectricity S1传感器
├── 15个独立色彩通道
├── 波长范围：400-700 nm
└── 马赛克像素级滤光片分布
```

## 技术优势

```
相比其他滤光技术：

✅ 更高的光谱分辨率
   └── 窄带响应，能精确分离波长

✅ 更优的稳定性
   └── 不受老化影响，长期稳定

✅ 更出色的光谱重建能力
   └── 高保真光谱信息

✅ 可芯片级集成
   └── 适合大规模量产

✅ 适合移动和消费电子
   └── 小型化、低成本潜力

✅ 可扩展性强
   └── 可适配不同图像传感器
```

## 技术指标

### 光谱性能

```
关键参数：

中心波长（CWL）：
├── 定义：透过率峰值对应的波长
├── 范围：可根据设计定制
├── Spectricity：400-700 nm范围
└── 精度：取决于层厚控制

半峰全宽（FWHM）：
├── 定义：透过率50%处的带宽
├── 典型值：1-10 nm
├── 越窄：光谱分辨率越高
└── 影响因素：反射率、腔长

峰值透过率：
├── 定义：中心波长的最大透过率
├── 典型值：50-90%
└── 影响因素：镜面反射率、吸收损耗
```

### 结构参数

```
设计要素：

镜面间距（d）：
├── 决定中心波长：λ = 2nd/m（n为折射率，m为级数）
├── 控制精度：纳米级
└── 调节方式：薄膜厚度控制

反射率（R）：
├── 决定干涉峰锐度
├── 越高：光谱分辨率越高
├── 典型值：80-99%
└── 实现方式：多层介质膜

腔数：
├── 单腔：简单，宽带
├── 双腔：较锐利，中等带宽
└── 多腔：极锐利，窄带宽
```

## 制造工艺

```
集成到CMOS传感器的工艺流程：

1. CMOS传感器制备
   └── 标准半导体工艺

2. 滤光层沉积
   ├── 物理气相沉积（PVD）
   ├── 化学气相沉积（CVD）
   └── 精确控制厚度

3. 马赛克图案化
   ├── 光刻工艺
   ├── 刻蚀工艺
   └── 像素级滤光片形成

4. 封装与测试
   ├── 光谱校准
   └── 质量检验

关键挑战：
├── 厚度控制精度（纳米级）
├── 层间均匀性
├── 与CMOS工艺兼容性
└── 成本控制
```

## vs 其他滤光技术

### 法布里-珀罗 vs 有机滤光片

| 维度 | 法布里-珀罗干涉 | 有机滤光片 |
|------|----------------|-----------|
| **光谱响应** | 窄带（<10 nm） | 宽带（>100 nm） |
| **稳定性** | 优异，不老化 | 较差，有机物老化 |
| **光谱分辨率** | 高 | 低 |
| **光谱重建能力** | 优秀 | 一般 |
| **数据处理需求** | 较低 | 较高（需大量处理） |
| **成本** | 中等 | 较低 |
| **适用场景** | 高性能应用 | 基础应用 |

### 法布里-珀罗 vs 导模共振（GMR）

| 维度 | 法布里-珀罗（Spectricity） | 导模共振（睿芯微） |
|------|---------------------------|-------------------|
| **原理** | 腔体干涉 | 波导模式耦合 |
| **光谱通道** | 5-15个 | 100-1000+个 |
| **光谱分辨率** | 中等 | 极高 |
| **集成方式** | 马赛克滤光片 | 片上光学结构 |
| **应用定位** | 消费级 | 医疗级 |
| **数据复杂度** | 中等 | 极高 |
| **代表产品** | S1传感器 | 高光谱CIS |

## 技术挑战

```
当前挑战：

⚠️ 制造精度要求极高
   └── 纳米级厚度控制

⚠️ 与CMOS工艺兼容性
   └── 需定制化工艺流程

⚠️ 成本控制
   └── 多层沉积成本较高

⚠️ 角度敏感性
   └── 入射角影响中心波长

⚠️ 温度稳定性
   └── 热膨胀导致波长漂移

解决方案：
├── 改进制造工艺
├── 材料优化
├── 温度补偿设计
└── 角度校正算法
```

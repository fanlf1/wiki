### 核心定位

```
甲方验收方案（摩托罗拉视角）：

目的：
├── 从采购方角度系统评估无创CGM方案

└── 供应链审核

### 验收原则

```
四大验收原则：

1️⃣ 以数据说话
   └── 所有指标需提供客观数据支撑

2️⃣ 可重复验证
   └── 测试方法可重复、可追溯

3️⃣ 风险可控
   └── 识别技术、合规、供应链风险

4️⃣ 符合监管
   └── 满足FDA/NMPA/CE等监管要求
```

## 🔬 第二部分：可操作的验收测试

### 测试一：准确性验证测试（Accuracy Validation）

#### 目的
验证MARD、ISO 15167、Clarke误差网格等关键准确性指标

#### 测试方案（甲方执行）

**1. 受试者招募**
```
样本量：N=20-30人
组成：
├── T1D患者：5人
├── T2D患者：10人
├── 健康人：5-10人
└── 覆盖不同肤色、年龄、BMI
```

**2. 测试流程**
```
第1天（校准）：
├── 空腹8小时后到达测试中心
├── 采集静脉血（参考值）
├── 佩戴无创CGM设备
└── 同时采集指尖血（校准）

第2-7天（连续监测）：
├── 每日4次指尖血对比（空腹、三餐后2小时）
├── 记录饮食、运动、药物
├── 记录任何设备问题
└── 第7天结束，摘除设备

特别测试：
├── OGTT测试（75g葡萄糖）
├── 运动测试（中等强度30分钟）
└── 低血糖诱导（仅在医疗监督下）
```

**3. 通过标准**
```
✅ 通过：
MARD ≤ 10%
Zone A+B ≥ 95%

⚠️ 条件通过：
MARD ≤ 12% 且 有合理解释
Zone A+B ≥ 90% 且 主要在Zone B

❌ 不通过：
MARD > 12%
Zone A+B < 90%
出现任何Zone D/E
```

---

### 测试二：重复性与精确性测试

#### 1. 重复性测试（同一样本）
```
方法：
1. 使用血糖模拟液（多个浓度水平）
   ├── 低血糖：3.9 mmol/L
   ├── 正常：5.6、7.0 mmol/L
   └── 高血糖：11.1、16.7 mmol/L

2. 每个浓度测量10次
   ├── 同一设备
   ├── 同一环境
   ├── 短时间内完成
   └── 不重新校准

3. 计算CV%（变异系数）
   CV% = (标准差 / 平均值) × 100%

通过标准：CV% ≤ 5%
```

#### 2. 设备间一致性
```
方法：
1. 准备5台设备
2. 同时测量同一样本（3个浓度）
3. 每台测量3次
4. 计算设备间CV%

通过标准：设备间CV% ≤ 8%
```

---

### 测试三：稳定性与漂移测试

#### 1. 长期稳定性测试
```
方法：
1. 选择10名受试者
2. 连续佩戴14天
3. 每日2次指尖血校准/验证
4. 记录每日漂移情况

通过标准：
- 第14天MARD ≤ 第1天MARD + 3%
- 无明显漂移趋势
```

#### 2. 温度漂移测试
```
方法：
1. 使用血糖模拟液
2. 在3个温度下测量：
   ├── 15℃（低温）
   ├── 25℃（常温）
   └── 35℃（高温）
3. 每个温度测量5次

通过标准：温度漂移 < 3%/10℃
```

---

### 测试四：实时性与延迟测试

#### 1. 响应时间测试
```
方法：
1. 受试者佩戴设备
2. 快速饮用含糖饮料（相当于50g葡萄糖）
3. 每1分钟测量一次指尖血
4. 记录设备响应时间

通过标准：响应时间 < 30秒
```

#### 2. 时间延迟分析
```
方法：
1. 进行OGTT测试（0, 15, 30, 60, 90, 120分钟）
2. 同时采集：
   ├── 静脉血（参考值）
   ├── 无创CGM数据
   └── 记录精确时间戳

3. 交叉相关分析
   - 计算不同时间偏移下的相关性
   - 找到最大相关性的时间偏移

通过标准：时间延迟 < 10分钟
```

---

### 测试五：安全性与生物相容性测试

#### 1. 皮肤刺激测试
```
方法：
1. 20名受试者
2. 佩戴设备24小时
3. 移除后观察皮肤反应
4. 评估标准：红斑、水肿、瘙痒等

评分标准（ISO 10993-10）：
0分：无反应
1分：轻微红斑
2分：明显红斑
3分：严重红斑+水肿
4分：严重反应

通过标准：所有受试者评分 ≤ 1分
```

#### 2. 温升测试
```
方法：
1. 受试者佩戴设备
2. 连续测量接触点温度
3. 测量时间：至少2小时
4. 环境温度：25℃

通过标准：最高温度 < 41℃
```

---

### 测试六：可靠性与寿命测试

#### 1. 传感器寿命测试
```
方法：
1. 使用模拟使用环境
2. 连续运行设备
3. 每日校准并记录MARD
4. 直到MARD > 15%或设备故障

通过标准：传感器寿命 ≥ 30天
优秀：传感器寿命 ≥ 90天
```

#### 2. MTBF测试
```
方法：
1. 抽取20台设备
2. 连续运行1000小时
3. 记录故障次数
4. 计算MTBF

MTBF = 总运行时间 / 故障次数

通过标准：MTBF ≥ 8760小时
```

## 📋 第三部分：验收流程与决策

### 阶段一：文件审核（1-2周）

#### 必须通过（否则不进入样机测试）
- [ ] 临床验证报告完整
- [ ] MARD ≤ 10%
- [ ] ISO 15167 Zone A+B ≥ 95%
- [ ] 电气安全报告齐全
- [ ] 知识产权清晰（无侵权风险）
- [ ] ISO 13485证书有效
- [ ] 批次记录可追溯

#### 决策标准
```
✅ 通过：
- 所有必要文件齐全
- 关键指标达标
- 无严重风险

⏳ 条件通过：
- 关键指标达标
- 部分文件缺失但有补充计划
- 非关键指标有偏差但有改进计划

❌ 不通过：
- 关键指标不达标
- 文件严重缺失
- 存在重大风险（IP、安全、合规）
```

---

### 阶段二：样机测试（4-6周）

#### 测试内容

**1. 实验室测试（1周）**
- 重复性测试
- 稳定性测试
- 温湿度漂移测试
- 电气安全测试
- 功能测试

**2. 人体测试（4周）**
- 招募20-30名受试者
- 连续监测14天
- 准确性验证
- 用户体验评估

**3. 可靠性测试（1周，并行）**
- 跌落、振动测试
- 寿命测试（加速）
- 环境适应性测试

#### 验收标准

**必须通过**
```
实验室测试：
✅ 重复性CV% ≤ 5%
✅ 14天MARD漂移 < 3%
✅ 温度漂移 < 3%/10℃
✅ 功能100%正常

人体测试：
✅ MARD ≤ 10%（90%置信区间上限 ≤ 12%）
✅ ISO 15167 Zone A+B ≥ 95%
✅ 无严重不良事件
✅ 用户满意度 ≥ 4/5

可靠性测试：
✅ 所有测试通过
✅ 无致命故障
```

✅ 批次记录完整可追溯

├── 传感器寿命 ≥ 30天
├── MTBF ≥ 8760小时
├── 产能 ≥ 10k/月
├── 知识产权无侵权风险
└── ISO 13485证书

⚠️ 警告项（需解释）：
├── MARD在8-10%之间
├── 部分非关键指标偏差
└── 供应商有单一来源
```

### 3. 验收流程的可操作性

```
验收流程的实操性：

六大可操作测试：
1. 准确性验证测试（20-30人，7-14天）
2. 重复性与精确性测试（实验室）
3. 稳定性与漂移测试（14天）
4. 实时性与延迟测试（OGTT）
5. 安全性与生物相容性测试（第三方）
6. 可靠性与寿命测试（加速老化）

配套检查清单：
├── 文件审核清单（150+项）
├── 实验室测试清单（标准化表格）
├── 人体测试清单（病例记录表）
├── 供应链审核清单（现场审核）
└── 风险评估清单（决策矩阵）
```

### 4. 风险管理意识

```
风险管理的全面性：

风险分类：
├── 技术风险
│   ├── MARD不达标
│   ├── 长期漂移
│   ├── 个体差异大
│   └── 批次差异
├── 合规风险
│   ├── 专利侵权
│   ├── 监管延迟
│   └── 数据隐私
└── 供应链风险
    ├── 产能不足
    ├── 单一来源
    └── 质量波动

风险等级：
🟢 低风险：总分 ≥ 85分，无关键风险项
🟡 中风险：总分 75-84分，有非关键风险项但可控
🔴 高风险：总分 < 75分，或有关键风险项
```

### 5. 对供应商的要求

```
对供应商的全面要求：

文件要求（150+项）：
├── 技术与性能报告（40+项）
├── 安全性与合规报告（30+项）
├── 质量体系报告（20+项）
├── 可靠性与寿命报告（20+项）
├── 知识产权报告（15+项）
├── 监管合规报告（20+项）
├── 供应链与量产报告（15+项）
└── 用户与市场报告（10+项）

质量体系要求：
├── ISO 13485认证（必须）
├── 批次记录可追溯（必须）
├── Yield ≥ 95%（必须）
├── 关键物料有二供（风险可控）
└── 供应商审核机制
```

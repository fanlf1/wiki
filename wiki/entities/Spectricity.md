## 核心技术

### 法鉑腔精密滤波技术

```
技术原理：
├── 基于法布里-珀罗干涉原理
├── 已获专利
├── 马赛克形式分布
├── 通过控制CMOS传感器上滤光层厚度决定中心波长
├── 每个像素都具备窄带光谱响应
├── 可扩展并适配任何图像传感器
└── 将多光谱成像带入超小型化领域

技术特点：
├── 高光谱分辨率（窄带响应）
├── 优异的稳定性
├── 出色的光谱重建能力
└── 可芯片级集成
```

### 技术路线选择

```
选择：干涉滤光片（法布里-珀罗）
原因：
✅ 更高的光谱分辨率
✅ 更优的稳定性
✅ 更出色的光谱重建能力
✅ 可在芯片上集成
✅ 适合大规模量产
✅ 非常适合移动和消费电子

vs 有机滤光片：
├── 有机：成本低，但精度低、稳定性差
└── 干涉：高性能、高稳定性、可量产
```

## 产品组合

### S1 多光谱图像传感器

```
定位：全球首款真正实现小型化且可大规模生产的多光谱图像传感器

技术规格：
├── 通道数：15个独立色彩通道
├── 波长范围：400-700 nm
├── 光谱响应：窄带
├── 滤光技术：法鉑腔精密滤波技术
├── 分布方式：马赛克像素级
├── 传感器基础：标准CMOS
└── 特点：可小型化、可大规模量产

vs 传统RGB：
├── RGB：3个通道
└── S1：15个通道（5倍数据量）
```

### S1-M 摄像头模组

```
定位：紧凑型摄像头模组

组件：
├── 专门设计的摄像头
├── 板载内存
└── S1 多光谱图像传感芯片

特点：
└── 经过生产校准，实现更准确一致的光谱响应
```

### S1-EVK2b 评估套件

```
定位：双相机单元开发套件

组件：
├── S1-M 多光谱摄像头模组
├── RGB 摄像头模组（共同封装）
└── NVIDIA Jetson 处理器

用途：
└── 应用程序开发
```

### S1-A 附件设备

```
定位：便携式数据采集设备

接口：USB

组件：
└── S1-M 摄像头模组

用途：
├── 与移动设备无缝配对
└── 数据采集和应用程序开发
```

# 多感科技（MIXOSENSE）

## 公司信息

- **全称**: 多感科技（MIXOSENSE）
- **行业**: 近红外光谱传感芯片
- **核心技术**: 近红外光谱 + 硅基分光芯片
- **官网**: www.mixosense.com

## 核心技术

### 近红外光谱方案

**技术路线**:
```
近红外光源 → 皮肤组织 → 硅基分光芯片 → InGaAs传感器 → 112通道光谱数据
                                    ↓
                检测葡萄糖分子O-H/C-H基团的光谱吸收特征（780-2500nm）
```

**技术原理**:
- 检测葡萄糖分子O-H和C-H基团的光谱吸收
- 透射/散射分析（通过组织间液推断血糖）
- 近红外波长范围：780-2500nm（重点900-1750nm）

### 硅基分光芯片架构

```
┌─────────────────────────────────────────────────┐
│              硅基分光芯片                         │
│  ┌─────────────────────────────────────────┐    │
│  │  Si Substrate（硅基底）                  │    │
│  │  Bottom DBR（底层布拉格反射镜）          │    │
│  │  Spacer（spacer厚度渐变）               │    │
│  │  Top DBR（顶层布拉格反射镜）             │    │
│  │                                          │    │
│  │  基于8吋硅光工艺平台                      │    │
│  │  薄膜厚度偏差控制<2%                     │    │
│  └─────────────────────────────────────────┘    │
└─────────────────────────────────────────────────┘
                        ↓
┌─────────────────────────────────────────────────┐
│       InGaAs PD 线阵传感器 + ROIC                │
│  • 12.5 μm像素                                   │
│  • 112通道近红外多光谱传感器                     │
│  • 透光区域 0.5×1.4 mm                           │
│  • QE符合设计预期                                │
└─────────────────────────────────────────────────┘
                        ↓
              112通道光谱数据 → 算法分析
```
